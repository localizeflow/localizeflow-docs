> **ဒေသတွင်းမှာ ကလုန်းဆွဲလုပ်ချင်ပါသလား?**

> ဒီ repository မှာ ၅၀ ကျော်သော ဘာသာစကားအတွက် ဘာသာပြန်ချက်များ ပါဝင်ပြီး ဒါကြောင့် ဒေါင်းလုပ် အရွယ်အစားကြီးစေပါတယ်။ ဘာသာပြန်ချက် မပါဘဲ ကလုန်းဆွဲချင်ရင် sparse checkout ကို အသုံးပြုပါ:
> ```bash
> git clone --filter=blob:none --sparse https://github.com/localizeflow/localizeflow-docs.git
> cd localizeflow-docs
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
> ဒါက သင့်ကို သင့်ကို သင်ယူမှုကို မြန်ဆန်စွာ ပြီးမြောက်စေဖို့ လိုအပ်သမျှကို ပေးပါတယ်။
<!-- CO-OP TRANSLATOR LANGUAGES TABLE END -->

Localizeflow ဟာ သင့်ရဲ့ စာရွက်စာတမ်းတွေကို ကိုယ်တိုင်းဘာသာပြန်ပြီး မူရင်းဖိုင်ပြောင်းလဲတိုင်း pull request များ ဖွင့်ပေးပါတယ်။  
ဒီ လမ်းညွှန်က GitHub App ကို ထည့်သွင်းနည်းနဲ့ ပထမဆုံး ဘာသာပြန်ချက်ကို ၂ မိနစ်အတွင်း တည်ဆောက်နည်းကို ပြသပေးပါတယ်။

> [!NOTE]
>
> Localizeflow က ယခုအခါ GitHub အခြေခံ စာရွက်စာတမ်း ပရောဂျက်များကိုသာ ပံ့ပိုးပေးပါတယ်  
> (ဥပမာ - AI for Beginners နဲ့ ပုံမှန် open-source repos များ)။  
> 
> Astro, Docusaurus, Hugo တို့လို ခေတ်မှီ စာရွက်စာတမ်း ဖွဲ့စည်းမှုများအတွက် ပံ့ပိုးမှုကို လက်ရှိ တွက်တင် ဆောင်ရွက်နေဆဲ ဖြစ်ပါတယ်။

## GitHub App မှာ ဂရုစိုက်နောက်ကာပြုရန်နှင့် ထည့်သွင်းရန်

၁။ **[localizeflow.com](https://localizeflow.com/)** ကိုသွားပါ။  
၂။ **Start with free trial** ကို ရွေးချယ်ပါ။  
   ![Select Start with free trial](../../../../translated images/my/select-start-with-free-trial.6c2e287133ff9c8b.webp)  
၃။ **Sign in with GitHub** ကို ရွေးချယ်ပါ။  
   ![Sign in with GitHub](../../../../translated images/my/select-sign-in-with-github.f2850ffdd49cc894.webp)  
၄။ သင့် GitHub အကောင့်ဖြင့် ဝင်ပါ။  
   ![GitHub login](../../../../translated images/my/github-login.02bbaee0270b292e.webp)  
၅။ Localizeflow GitHub App ကို ထည့်သွင်းချင်သော အကောင့်ကို (ကိုယ်ပိုင်အကောင့်သို့မဟုတ် လူမှုအဖွဲ့အစည်း တစ်ခု) ရွေးချယ်ပါ။  
   ![Select account](../../../../translated images/my/select-which-account-to-use.7050f5ed0b773bb0.webp)  
၆။ Localizeflow ရဲ့ ဝင်ရောက်ခွင့်ပြုလိုသော repositories များကို ရွေးချယ်ပြီး **Save** နှိပ်ပါ။  
   ![Select repo and save](../../../../translated images/my/select-repo-and-save.5a95ae288aefec6e.webp)  
၇။ ပြန်လည် Localizeflow စာမျက်နှာသို့သွားရောက်မည် ဖြစ်သည်။

> [!TIP]
> နောက်မှ repository တွေအများကြီး ထည့်လိုပါက header မှာ သင့်အကောင့်ကိုရွေးပြီး **+ Add more repositories** ကိုရွေးယူပါ။  
> ![Add more repositories](../../../../translated images/my/add-more-repo.2ca5154473aaaafb.webp)

## သင့် repositories များကို Localizeflow နှင့် ချိတ်ဆက်ရန်

၁။ Localizeflow စာမျက်နှာမူလအကြောင်းအရာ မှာ **+ Connect repositories** ကို ရွေးပါ။  
   ![Select connect repositories](../../../../translated images/my/select-connect-repos.8ac6f96f77dcc62c.webp)

၂။ သင်ချိတ်ဆက်လိုသော ထည့်သွင်းပြီးသား repository တစ်ခုကို ရွေးပြီး **Save** ကို နှိပ်ပါ။  
   ![Select repository](../../../../translated images/my/select-repo.dce94db722b44cf3.webp)

၃။ သင့်ချိတ်ဆက်ထားသော repositories များကို ယခု Home စာမျက်နှာနဲ့ Repositories စာမျက်နှာ နှစ်ခုလုံးမှာ မြင်ရပါမည်။  
   ![Connected repositories](../../../../translated images/my/repo-connected.9e5c21ee789fdcaa.webp)

## အလိုအလျောက် ဘာသာပြန်ခြင်း စတင်ရန်

၁။ သင့်ချိတ်ဆက်ထားသော repository ကို ရွေးချယ်ပါ။  
   ![Select repository](../../../../translated images/my/select-repo-to-detail.55e53233531f8518.webp)

၂။ Repository နောက်ကြောင်းစာမျက်နှာ၏ အောက်ခြေတွင် **Edit** ကို ရွေးပါ။  
   ![Select edit](../../../../translated images/my/select-edit.225184c8c46d7001.webp)

၃။ သင်၏ ဘာသာပြန်ရန် ဆက်တင်များကို ပြင်ဆင်ပါ — သတ်မှတ်ထားသော branch (ပုံမှန်: `main`), ရည်မှန်းဘာသာစကားများနှင့် မူရင်းဘာသာစကား (ပုံမှန်: `en`)။ **Save** ကို နှိပ်ပါ။  
   ![Configure translation settings](../../../../translated images/my/configuration.ab55d0f8bab5711b.webp)

၄။ **Start & Automate** ကို ရွေးချယ်ပါ။  
   Localizeflow သည် ယခုမှ စ၍ သင့်စာရွက်စာတမ်းများကို အလိုအလျောက် ဘာသာပြန်ပြီး မူရင်းပြောင်းလဲမှုတိုင်းတွင် pull requests ဖြင့် ဖွင့်ပေးပါမည်။  
   ![Start & Automate](../../../../translated images/my/select-automate.ff50c8d913b35a03.webp)
