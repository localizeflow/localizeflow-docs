Localizeflow သည် သင့်စာရွက်စာတမ်းများကို အလိုအလျောက် ဘာသာပြန်ပေးပြီး မူရင်းဖိုင်ပြောင်းလဲသည့်အခါ အလိုအလျောက် pull request များဖွင့်ပေးပါသည်။  
ဤလမ်းညွှန်တွင် GitHub App ကို မည်သို့တပ်ဆင်ပြီး ၂ မိနစ်အတွင်း ပထမဆုံး ဘာသာပြန်မှုကို မည်သို့စတင်ရမည်ကို ပြသထားသည်။

---

## GitHub App သို့ လက်မှတ်ထိုးပြီး တပ်ဆင်ခြင်း

1. **[localizeflow.com](https://localizeflow.com/)** သို့ သွားပါ။
2. **Start with free trial** ကို ရွေးချယ်ပါ။  
   ![Select Start with free trial](/images/select-start-with-free-trial.png)
3. **Sign in with GitHub** ကို ရွေးချယ်ပါ။  
   ![Sign in with GitHub](/images/select-sign-in-with-github.png)
4. သင့် GitHub အကောင့်ဖြင့် လက်မှတ်ထိုးဝင်ပါ။  
   ![GitHub login](/images/github-login.png)
5. Localizeflow GitHub App ကို တပ်ဆင်လိုသည့် အကောင့်ကို ရွေးချယ်ပါ — ကိုယ်ပိုင်အကောင့် သို့မဟုတ် သင်စီမံခန့်ခွဲသော အဖွဲ့အစည်းအကောင့်။  
   ![Select account](/images/select-which-account-to-use.png)
6. Localizeflow သုံးရန်လိုသော repository များကို ရွေးပြီး **Save** ကိုနှိပ်ပါ။  
   ![Select repo and save](/images/select-repo-and-save.png)
7. Localizeflow မူလစာမျက်နှာသို့ ပြန်လည်သွားပါလိမ့်မည်။

> [!TIP]
> နောက်ပိုင်းတွင် repository များ ပိုမိုထည့်လိုပါက header တွင် သင့်အကောင့်ကို ရွေးပြီး **+ Add more repositories** ကိုနှိပ်ပါ။  
> ![Add more repositories](/images/add-more-repo.png)

---

## သင့် repository များကို Localizeflow နှင့် ချိတ်ဆက်ခြင်း

1. Localizeflow မူလစာမျက်နှာတွင် **+ Connect repositories** ကို ရွေးချယ်ပါ။  
   ![Select connect repositories](/images/select-connect-repos.png)

2. တပ်ဆင်ပြီးသား repository များထဲမှ ချိတ်ဆက်လိုသည့် repository ကို ရွေးပြီး **Save** ကိုနှိပ်ပါ။  
   ![Select repository](/images/select-repo.png)

3. ချိတ်ဆက်ပြီးသော repository များသည် မူလစာမျက်နှာနှင့် Repositories စာမျက်နှာတွင် ပြသပါလိမ့်မည်။  
   ![Connected repositories](/images/repo-connected.png)

---

## အလိုအလျောက် ဘာသာပြန်မှု စတင်ခြင်း

1. ချိတ်ဆက်ပြီး repository ကို ရွေးချယ်ပါ။  
   ![Select repository](/images/select-repo-to-detail.png)

2. Repository အသေးစိတ်စာမျက်နှာတွင် အောက်ဆုံးရှိ **Edit** ကို နှိပ်ပါ။  
   ![Select edit](/images/select-edit.png)

3. ဘာသာပြန်မှု ဆက်တင်များကို ပြင်ဆင်ပါ — ရည်မှန်း branch (ပုံမှန်: `main`), ရည်မှန်းဘာသာစကားများနှင့် မူရင်းဘာသာစကား (ပုံမှန်: `en`)။ **Save** ကိုနှိပ်ပါ။  
   ![Configure translation settings](/images/configuration.png)

4. **Start & Automate** ကို ရွေးချယ်ပါ။  
   Localizeflow သည် မူရင်းဖိုင်ပြောင်းလဲသည့်အခါ အလိုအလျောက် သင့်စာရွက်စာတမ်းများကို ဘာသာပြန်ပြီး pull request များဖွင့်ပေးပါလိမ့်မည်။  
   ![Start & Automate](/images/select-automate.png)
