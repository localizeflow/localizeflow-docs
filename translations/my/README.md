Localizeflow သည် သင့်စာရွက်စာတမ်းများကို အလိုအလျောက်ဘာသာပြန်ပြီး မူရင်းဖိုင်ပြောင်းလဲသည့်အခါတွင် pull requests များဖွင့်ပေးသည်။  
ဤလမ်းညွှန်သည် GitHub App ကို မည်သို့တပ်ဆင်ပြီး ၂ မိနစ်အတွင်း ပထမဆုံးဘာသာပြန်မှုကို မည်သို့ပြုလုပ်ရမည်ကို ပြသသည်။

> [!NOTE]
>
> Localizeflow သည် လက်ရှိတွင် GitHub အခြေပြု စာရွက်စာတမ်းပရောဂျက်များကို ထောက်ပံ့သည်  
> (ဥပမာ - AI for Beginners နှင့် အများဆုံး စံပြ open-source repos များ)။  
> 
> Astro, Docusaurus, နှင့် Hugo ကဲ့သို့ ခေတ်မီစာရွက်စာတမ်းဖွဲ့စည်းမှု frameworks များအတွက် ထောက်ပံ့မှုကို လက်ရှိ တိုးတက်အဆင့်တွင် ရှိသည်။

---

## GitHub App သို့ လက်မှတ်ထိုး၍ တပ်ဆင်ခြင်း

1. **[localizeflow.com](https://localizeflow.com/)** သို့ သွားပါ။
2. **Start with free trial** ကို ရွေးချယ်ပါ။  
   ![Select Start with free trial](/images/select-start-with-free-trial.png)
3. **Sign in with GitHub** ကို ရွေးချယ်ပါ။  
   ![Sign in with GitHub](/images/select-sign-in-with-github.png)
4. သင့် GitHub အကောင့်ဖြင့် လက်မှတ်ထိုးဝင်ပါ။  
   ![GitHub login](/images/github-login.png)
5. Localizeflow GitHub App ကို တပ်ဆင်လိုသည့် အကောင့်ကို ရွေးချယ်ပါ — သင့်ကိုယ်ပိုင်အကောင့် သို့မဟုတ် သင်စီမံခန့်ခွဲသော အဖွဲ့အစည်း။  
   ![Select account](/images/select-which-account-to-use.png)
6. Localizeflow သည် ဝင်ရောက်ခွင့်ရရှိစေရန် သင်လိုချင်သော repositories များကို ရွေးချယ်ပြီး **Save** ကိုနှိပ်ပါ။  
   ![Select repo and save](/images/select-repo-and-save.png)
7. သင်သည် Localizeflow မူလစာမျက်နှာသို့ ပြန်လည်သွားရောက်မည်။

> [!TIP]
> နောက်ပိုင်းတွင် repository များ ပိုမိုထည့်သွင်းလိုပါက header တွင် သင့်အကောင့်ကို ရွေးချယ်ပြီး **+ Add more repositories** ကို ရွေးချယ်ပါ။  
> ![Add more repositories](/images/add-more-repo.png)

---

## သင့် repositories များကို Localizeflow နှင့် ချိတ်ဆက်ခြင်း

1. Localizeflow မူလစာမျက်နှာတွင် **+ Connect repositories** ကို ရွေးချယ်ပါ။  
   ![Select connect repositories](/images/select-connect-repos.png)

2. တပ်ဆင်ပြီးသား repositories များထဲမှ ချိတ်ဆက်လိုသည့် repository တစ်ခုကို ရွေးချယ်ပြီး **Save** ကိုနှိပ်ပါ။  
   ![Select repository](/images/select-repo.png)

3. ချိတ်ဆက်ပြီးသော repositories များသည် မူလစာမျက်နှာနှင့် Repositories စာမျက်နှာတွင် ပြသမည်။  
   ![Connected repositories](/images/repo-connected.png)

---

## အလိုအလျောက်ဘာသာပြန်မှု စတင်ခြင်း

1. သင်ချိတ်ဆက်ထားသော repository ကို ရွေးချယ်ပါ။  
   ![Select repository](/images/select-repo-to-detail.png)

2. Repository အသေးစိတ်စာမျက်နှာတွင် အောက်ခြေမှ **Edit** ကို ရွေးချယ်ပါ။  
   ![Select edit](/images/select-edit.png)

3. သင့်ဘာသာပြန်မှု ဆက်တင်များကို ပြင်ဆင်ပါ — ရည်မှန်း branch (ပုံမှန်: `main`), ရည်မှန်းဘာသာစကားများနှင့် မူရင်းဘာသာစကား (ပုံမှန်: `en`)။ **Save** ကိုနှိပ်ပါ။  
   ![Configure translation settings](/images/configuration.png)

4. **Start & Automate** ကို ရွေးချယ်ပါ။  
   Localizeflow သည် သင့်စာရွက်စာတမ်းများကို အလိုအလျောက်ဘာသာပြန်ပြီး မူရင်းပြောင်းလဲမှုတိုင်းတွင် pull requests ဖွင့်ပေးမည်။  
   ![Start & Automate](/images/select-automate.png)
