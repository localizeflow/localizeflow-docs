Localizeflow traduz automaticamente sua documentação e abre pull requests sempre que o arquivo fonte é alterado.  
Este guia mostra como instalar o GitHub App e executar sua primeira tradução em menos de 2 minutos.


> [!NOTE]
>
> Atualmente, o Localizeflow suporta projetos de documentação baseados no GitHub  
> (por exemplo: AI for Beginners e a maioria dos repositórios open-source padrão).  
> 
> O suporte para frameworks modernos de documentação como Astro, Docusaurus e Hugo  
> está em desenvolvimento ativo.


---

## Faça login e instale o GitHub App

1. Visite **[localizeflow.com](https://localizeflow.com/)**.
2. Selecione **Start with free trial**.
   ![Select Start with free trial](/images/select-start-with-free-trial.png)
3. Selecione **Sign in with GitHub**.  
   ![Sign in with GitHub](/images/select-sign-in-with-github.png)
4. Faça login com sua conta do GitHub.  
   ![GitHub login](/images/github-login.png)
5. Escolha a conta onde deseja instalar o Localizeflow GitHub App — sua conta pessoal ou uma organização que você gerencia.  
   ![Select account](/images/select-which-account-to-use.png)
6. Selecione os repositórios que deseja que o Localizeflow acesse e depois escolha **Save**.  
   ![Select repo and save](/images/select-repo-and-save.png)
7. Você será redirecionado para a página inicial do Localizeflow.

> [!TIP]
> Para adicionar mais repositórios depois, selecione sua conta no cabeçalho e escolha **+ Add more repositories**.  
> ![Add more repositories](/images/add-more-repo.png)

---

## Conecte seus repositórios ao Localizeflow

1. Na página inicial do Localizeflow, selecione **+ Connect repositories**.  
   ![Select connect repositories](/images/select-connect-repos.png)

2. Escolha um dos repositórios instalados que deseja conectar e selecione **Save**.  
   ![Select repository](/images/select-repo.png)

3. Seus repositórios conectados agora aparecerão tanto na página Home quanto na página Repositories.  
   ![Connected repositories](/images/repo-connected.png)

---

## Comece a tradução automática

1. Selecione o repositório que você acabou de conectar.  
   ![Select repository](/images/select-repo-to-detail.png)

2. Na página de detalhes do repositório, selecione **Edit** na parte inferior.  
   ![Select edit](/images/select-edit.png)

3. Configure suas configurações de tradução — branch alvo (padrão: `main`), idiomas alvo e idioma fonte (padrão: `en`). Selecione **Save**.  
   ![Configure translation settings](/images/configuration.png)

4. Selecione **Start & Automate**.  
   O Localizeflow agora irá traduzir automaticamente sua documentação e abrir pull requests sempre que a fonte for alterada.  
   ![Start & Automate](/images/select-automate.png)
