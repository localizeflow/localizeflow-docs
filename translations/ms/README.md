Localizeflow secara automatik menterjemah dokumentasi anda dan membuka permintaan tarik setiap kali fail sumber berubah.  
Panduan ini menunjukkan cara memasang Aplikasi GitHub dan menjalankan terjemahan pertama anda dalam masa kurang dari 2 minit.


> [!NOTE]
>
> Localizeflow kini menyokong projek dokumentasi berasaskan GitHub
> (contohnya: AI for Beginners dan kebanyakan repositori sumber terbuka standard).  
> 
> Sokongan untuk rangka kerja dokumentasi moden seperti Astro, Docusaurus, dan Hugo  
> sedang dalam pembangunan aktif.


---

## Log masuk dan pasang Aplikasi GitHub

1. Lawati **[localizeflow.com](https://localizeflow.com/)**.
2. Pilih **Mulakan dengan percubaan percuma**.
   ![Select Start with free trial](/images/select-start-with-free-trial.png)
3. Pilih **Log masuk dengan GitHub**.  
   ![Sign in with GitHub](/images/select-sign-in-with-github.png)
4. Log masuk dengan akaun GitHub anda.  
   ![GitHub login](/images/github-login.png)
5. Pilih akaun di mana anda ingin memasang Aplikasi GitHub Localizeflow — akaun peribadi anda atau organisasi yang anda uruskan.  
   ![Select account](/images/select-which-account-to-use.png)
6. Pilih repositori yang anda mahu Localizeflow akses, kemudian pilih **Simpan**.  
   ![Select repo and save](/images/select-repo-and-save.png)
7. Anda akan dialihkan ke halaman utama Localizeflow.

> [!TIP]
> Untuk menambah lebih banyak repositori kemudian, pilih akaun anda di tajuk dan pilih **+ Tambah lebih banyak repositori**.  
> ![Add more repositories](/images/add-more-repo.png)

---

## Sambungkan repositori anda ke Localizeflow

1. Di halaman utama Localizeflow, pilih **+ Sambungkan repositori**.  
   ![Select connect repositories](/images/select-connect-repos.png)

2. Pilih salah satu repositori yang dipasang yang anda mahu sambungkan dan pilih **Simpan**.  
   ![Select repository](/images/select-repo.png)

3. Repositori yang disambungkan kini akan muncul di halaman Utama dan halaman Repositori.  
   ![Connected repositories](/images/repo-connected.png)

---

## Mulakan terjemahan automatik

1. Pilih repositori yang baru anda sambungkan.  
   ![Select repository](/images/select-repo-to-detail.png)

2. Di halaman butiran repositori, pilih **Edit** di bahagian bawah.  
   ![Select edit](/images/select-edit.png)

3. Konfigurasikan tetapan terjemahan anda — cawangan sasaran (lalai: `main`), bahasa sasaran, dan bahasa sumber (lalai: `en`). Pilih **Simpan**.  
   ![Configure translation settings](/images/configuration.png)

4. Pilih **Mula & Automatikkan**.  
   Localizeflow kini akan secara automatik menterjemah dokumentasi anda dan membuka permintaan tarik setiap kali sumber berubah.  
   ![Start & Automate](/images/select-automate.png)
