> **Mas gusto mo bang i-Clone nang Lokal?**

> Kasama sa repositoryong ito ang 50+ na pagsasalin sa iba't ibang wika na malaki ang dagdag sa laki ng iyong id-download. Para mag-clone nang wala ang mga pagsasalin, gamitin ang sparse checkout:
> ```bash
> git clone --filter=blob:none --sparse https://github.com/localizeflow/localizeflow-docs.git
> cd localizeflow-docs
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
> Ito ay nagbibigay sa iyo ng lahat ng kailangan upang matapos ang kurso nang mas mabilis ang pag-download.

Awtomatikong isinasalin ng Localizeflow ang iyong dokumentasyon at nagbubukas ng mga pull request kapag binago ang source file.  
Ipinapakita ng gabay na ito kung paano i-install ang GitHub App at patakbuhin ang iyong unang pagsasalin sa loob ng 2 minuto.

> [!NOTE]
>
> Suportado ngayon ng Localizeflow ang mga proyekto ng dokumentasyon na naka-GitHub  
> (halimbawa: AI for Beginners at karamihan ng mga karaniwang open-source repos).  
> 
> Aktibo ang pagde-develop ng suporta para sa mga modernong documentation framework tulad ng Astro, Docusaurus, at Hugo.

## Mag-sign in at i-install ang GitHub App

1. Bisitahin ang **[localizeflow.com](https://localizeflow.com/)**.
2. Piliin ang **Start with free trial**.
   ![Select Start with free trial](../../../../translated images/tl/select-start-with-free-trial.6c2e287133ff9c8b.webp)
3. Piliin ang **Sign in with GitHub**.  
   ![Sign in with GitHub](../../../../translated images/tl/select-sign-in-with-github.f2850ffdd49cc894.webp)
4. Mag-sign in gamit ang iyong GitHub account.  
   ![GitHub login](../../../../translated images/tl/github-login.02bbaee0270b292e.webp)
5. Piliin ang account kung saan mo gustong i-install ang Localizeflow GitHub App — personal mong account o isang organisasyon na iyong pinamamahalaan.  
   ![Select account](../../../../translated images/tl/select-which-account-to-use.7050f5ed0b773bb0.webp)
6. Piliin ang mga repositoryo na gusto mong ma-access ng Localizeflow, pagkatapos ay piliin ang **Save**.  
   ![Select repo and save](../../../../translated images/tl/select-repo-and-save.5a95ae288aefec6e.webp)
7. Mai-redirect ka sa home page ng Localizeflow.

> [!TIP]
> Para magdagdag ng higit pang mga repositoryo mamaya, piliin ang iyong account sa header at piliin ang **+ Add more repositories**.  
> ![Add more repositories](../../../../translated images/tl/add-more-repo.2ca5154473aaaafb.webp)

## Ikonekta ang iyong mga repositoryo sa Localizeflow

1. Sa home page ng Localizeflow, piliin ang **+ Connect repositories**.  
   ![Select connect repositories](../../../../translated images/tl/select-connect-repos.8ac6f96f77dcc62c.webp)

2. Piliin ang isa sa mga naka-install na repositoryo na gusto mong ikonekta at piliin ang **Save**.  
   ![Select repository](../../../../translated images/tl/select-repo.dce94db722b44cf3.webp)

3. Makikita mo na ngayon ang mga nakonektang repositoryo sa parehong Home page at sa Repositories page.  
   ![Connected repositories](../../../../translated images/tl/repo-connected.9e5c21ee789fdcaa.webp)

---

## Simulan ang awtomatikong pagsasalin

1. Piliin ang repositoryong katatapos mo lang ikonekta.  
   ![Select repository](../../../../translated images/tl/select-repo-to-detail.55e53233531f8518.webp)

2. Sa detalye ng repositoryo, piliin ang **Edit** sa ibaba.  
   ![Select edit](../../../../translated images/tl/select-edit.225184c8c46d7001.webp)

3. I-configure ang iyong mga setting ng pagsasalin — target branch (default: `main`), target na mga wika, at source na wika (default: `en`). Piliin ang **Save**.  
   ![Configure translation settings](../../../../translated images/tl/configuration.ab55d0f8bab5711b.webp)

4. Piliin ang **Start & Automate**.  
   Awtomatikong isasalin ng Localizeflow ang iyong dokumentasyon at magbubukas ng mga pull request kapag may pagbabago sa source.  
   ![Start & Automate](../../../../translated images/tl/select-automate.ff50c8d913b35a03.webp)
