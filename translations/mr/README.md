Localizeflow आपले दस्तऐवजीकरण स्वयंचलितपणे भाषांतरित करते आणि स्रोत फाइल बदलल्यावर पुल विनंत्या उघडते.  
हा मार्गदर्शक आपल्याला GitHub अॅप कसे स्थापित करायचे आणि 2 मिनिटांपेक्षा कमी वेळात आपले पहिले भाषांतर कसे चालवायचे हे दर्शवितो.


> [!NOTE]
>
> Localizeflow सध्या GitHub-आधारित दस्तऐवजीकरण प्रकल्पांना समर्थन देते  
> (उदाहरणार्थ: AI for Beginners आणि बहुतेक मानक ओपन-सोर्स रेपॉजिटरीज).  
> 
> Astro, Docusaurus, आणि Hugo सारख्या आधुनिक दस्तऐवजीकरण फ्रेमवर्कसाठी समर्थन  
> सक्रिय विकासात आहे.


---

## साइन इन करा आणि GitHub अॅप स्थापित करा

1. भेट द्या **[localizeflow.com](https://localizeflow.com/)**.
2. निवडा **Start with free trial**.
   ![Select Start with free trial](/images/select-start-with-free-trial.png)
3. निवडा **Sign in with GitHub**.  
   ![Sign in with GitHub](/images/select-sign-in-with-github.png)
4. आपल्या GitHub खात्याने साइन इन करा.  
   ![GitHub login](/images/github-login.png)
5. Localizeflow GitHub अॅप कुठल्या खात्यावर स्थापित करायचा आहे ते निवडा — आपले वैयक्तिक खाते किंवा आपण व्यवस्थापित करणारी संस्था.  
   ![Select account](/images/select-which-account-to-use.png)
6. Localizeflow ला प्रवेश द्यायच्या रेपॉजिटरीज निवडा, नंतर **Save** निवडा.  
   ![Select repo and save](/images/select-repo-and-save.png)
7. आपण Localizeflow मुख्य पृष्ठावर पुनर्निर्देशित केले जाल.

> [!TIP]
> नंतर अधिक रेपॉजिटरीज जोडण्यासाठी, शीर्षलेखात आपले खाते निवडा आणि **+ Add more repositories** निवडा.  
> ![Add more repositories](/images/add-more-repo.png)

---

## आपले रेपॉजिटरीज Localizeflow शी कनेक्ट करा

1. Localizeflow मुख्य पृष्ठावर, निवडा **+ Connect repositories**.  
   ![Select connect repositories](/images/select-connect-repos.png)

2. स्थापित केलेल्या रेपॉजिटरीजपैकी एक निवडा ज्याला आपण कनेक्ट करू इच्छिता आणि **Save** निवडा.  
   ![Select repository](/images/select-repo.png)

3. आपले कनेक्ट केलेले रेपॉजिटरीज आता मुख्य पृष्ठावर आणि रेपॉजिटरीज पृष्ठावर दिसतील.  
   ![Connected repositories](/images/repo-connected.png)

---

## स्वयंचलित भाषांतर सुरू करा

1. आपण नुकतेच कनेक्ट केलेले रेपॉजिटरी निवडा.  
   ![Select repository](/images/select-repo-to-detail.png)

2. रेपॉजिटरी तपशील पृष्ठावर, खालील बाजूला **Edit** निवडा.  
   ![Select edit](/images/select-edit.png)

3. आपले भाषांतर सेटिंग्ज कॉन्फिगर करा — लक्ष्य शाखा (डिफॉल्ट: `main`), लक्ष्य भाषा, आणि स्रोत भाषा (डिफॉल्ट: `en`). **Save** निवडा.  
   ![Configure translation settings](/images/configuration.png)

4. निवडा **Start & Automate**.  
   Localizeflow आता आपले दस्तऐवजीकरण स्वयंचलितपणे भाषांतरित करेल आणि स्रोत बदलल्यावर पुल विनंत्या उघडेल.  
   ![Start & Automate](/images/select-automate.png)
