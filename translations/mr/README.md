Localizeflow आपोआप तुमची दस्तऐवज अनुवादित करते आणि स्रोत फाइल बदलल्यावर पुल विनंत्या उघडते.  
हा मार्गदर्शक तुम्हाला GitHub अॅप कसे इन्स्टॉल करायचे आणि २ मिनिटांत तुमचा पहिला अनुवाद कसा चालवायचा हे दाखवतो.

---

## साइन इन करा आणि GitHub अॅप इन्स्टॉल करा

1. भेट द्या **[localizeflow.com](https://localizeflow.com/)**.
2. निवडा **Start with free trial**.
   ![Select Start with free trial](/images/select-start-with-free-trial.png)
3. निवडा **Sign in with GitHub**.  
   ![Sign in with GitHub](/images/select-sign-in-with-github.png)
4. तुमच्या GitHub खात्याने साइन इन करा.  
   ![GitHub login](/images/github-login.png)
5. Localizeflow GitHub अॅप कुठल्या खात्यावर इन्स्टॉल करायचे ते निवडा — तुमचे वैयक्तिक खाते किंवा तुम्ही व्यवस्थापित करत असलेली संस्था.  
   ![Select account](/images/select-which-account-to-use.png)
6. Localizeflow ला प्रवेश द्यायच्या रेपॉजिटरीज निवडा, नंतर **Save** वर क्लिक करा.  
   ![Select repo and save](/images/select-repo-and-save.png)
7. तुम्हाला Localizeflow मुख्य पृष्ठावर पुनर्निर्देशित केले जाईल.

> [!TIP]
> नंतर अधिक रेपॉजिटरीज जोडण्यासाठी, हेडरमधील तुमचे खाते निवडा आणि **+ Add more repositories** निवडा.  
> ![Add more repositories](/images/add-more-repo.png)

---

## तुमच्या रेपॉजिटरीजना Localizeflow शी जोडा

1. Localizeflow मुख्य पृष्ठावर, निवडा **+ Connect repositories**.  
   ![Select connect repositories](/images/select-connect-repos.png)

2. इन्स्टॉल केलेल्या रेपॉजिटरीजपैकी एक निवडा ज्याला तुम्हाला जोडायचे आहे आणि **Save** निवडा.  
   ![Select repository](/images/select-repo.png)

3. तुमच्या जोडलेल्या रेपॉजिटरीज आता मुख्य पृष्ठावर आणि रेपॉजिटरीज पृष्ठावर दिसतील.  
   ![Connected repositories](/images/repo-connected.png)

---

## स्वयंचलित अनुवाद सुरू करा

1. नुकतीच जोडलेली रेपॉजिटरी निवडा.  
   ![Select repository](/images/select-repo-to-detail.png)

2. रेपॉजिटरी तपशील पृष्ठावर, खालील बाजूला **Edit** निवडा.  
   ![Select edit](/images/select-edit.png)

3. तुमचे अनुवाद सेटिंग्ज कॉन्फिगर करा — लक्ष्य शाखा (डिफॉल्ट: `main`), लक्ष्य भाषा, आणि स्रोत भाषा (डिफॉल्ट: `en`). नंतर **Save** निवडा.  
   ![Configure translation settings](/images/configuration.png)

4. निवडा **Start & Automate**.  
   Localizeflow आता तुमची दस्तऐवज आपोआप अनुवादित करेल आणि स्रोत बदलल्यावर पुल विनंत्या उघडेल.  
   ![Start & Automate](/images/select-automate.png)
