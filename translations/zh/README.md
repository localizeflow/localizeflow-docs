Localizeflow 会自动翻译您的文档，并在源文件发生更改时自动发起拉取请求。  
本指南将教您如何安装 GitHub 应用，并在不到 2 分钟内完成首次翻译。

---

## 登录并安装 GitHub 应用

1. 访问 **[localizeflow.com](https://localizeflow.com/)**。
2. 选择 **开始免费试用**。  
   ![选择开始免费试用](/images/select-start-with-free-trial.png)
3. 选择 **使用 GitHub 登录**。  
   ![使用 GitHub 登录](/images/select-sign-in-with-github.png)
4. 使用您的 GitHub 账号登录。  
   ![GitHub 登录](/images/github-login.png)
5. 选择您想安装 Localizeflow GitHub 应用的账户——个人账户或您管理的组织。  
   ![选择账户](/images/select-which-account-to-use.png)
6. 选择您希望 Localizeflow 访问的仓库，然后点击 **保存**。  
   ![选择仓库并保存](/images/select-repo-and-save.png)
7. 您将被重定向到 Localizeflow 首页。

> [!TIP]
> 如果以后想添加更多仓库，可以点击页面顶部的账户名，选择 **+ 添加更多仓库**。  
> ![添加更多仓库](/images/add-more-repo.png)

---

## 将仓库连接到 Localizeflow

1. 在 Localizeflow 首页，点击 **+ 连接仓库**。  
   ![选择连接仓库](/images/select-connect-repos.png)

2. 选择一个已安装的仓库进行连接，然后点击 **保存**。  
   ![选择仓库](/images/select-repo.png)

3. 您连接的仓库现在会显示在首页和仓库页面。  
   ![已连接的仓库](/images/repo-connected.png)

---

## 开始自动翻译

1. 选择刚刚连接的仓库。  
   ![选择仓库](/images/select-repo-to-detail.png)

2. 在仓库详情页底部，点击 **编辑**。  
   ![选择编辑](/images/select-edit.png)

3. 配置翻译设置——目标分支（默认：`main`）、目标语言和源语言（默认：`en`）。点击 **保存**。  
   ![配置翻译设置](/images/configuration.png)

4. 点击 **开始并自动化**。  
   Localizeflow 现在会自动翻译您的文档，并在源文件更改时自动发起拉取请求。  
   ![开始并自动化](/images/select-automate.png)
