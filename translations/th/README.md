Localizeflow จะแปลเอกสารของคุณโดยอัตโนมัติและเปิด pull requests ทุกครั้งที่ไฟล์ต้นฉบับมีการเปลี่ยนแปลง  
คู่มือนี้จะแสดงวิธีติดตั้ง GitHub App และเริ่มการแปลครั้งแรกของคุณภายในเวลาไม่ถึง 2 นาที

---

## ลงชื่อเข้าใช้และติดตั้ง GitHub App

1. ไปที่ **[localizeflow.com](https://localizeflow.com/)**  
2. เลือก **เริ่มต้นทดลองใช้ฟรี**  
   ![เลือก เริ่มต้นทดลองใช้ฟรี](/images/select-start-with-free-trial.png)  
3. เลือก **ลงชื่อเข้าใช้ด้วย GitHub**  
   ![ลงชื่อเข้าใช้ด้วย GitHub](/images/select-sign-in-with-github.png)  
4. ลงชื่อเข้าใช้ด้วยบัญชี GitHub ของคุณ  
   ![เข้าสู่ระบบ GitHub](/images/github-login.png)  
5. เลือกบัญชีที่คุณต้องการติดตั้ง Localizeflow GitHub App — บัญชีส่วนตัวของคุณหรือองค์กรที่คุณดูแล  
   ![เลือกบัญชี](/images/select-which-account-to-use.png)  
6. เลือก repository ที่คุณต้องการให้ Localizeflow เข้าถึง จากนั้นเลือก **บันทึก**  
   ![เลือก repo และบันทึก](/images/select-repo-and-save.png)  
7. คุณจะถูกเปลี่ยนเส้นทางไปยังหน้าแรกของ Localizeflow

> [!TIP]
> หากต้องการเพิ่ม repository เพิ่มเติมในภายหลัง ให้เลือกบัญชีของคุณที่ส่วนหัวแล้วเลือก **+ เพิ่ม repository เพิ่มเติม**  
> ![เพิ่ม repository เพิ่มเติม](/images/add-more-repo.png)

---

## เชื่อมต่อ repository ของคุณกับ Localizeflow

1. ที่หน้าแรกของ Localizeflow เลือก **+ เชื่อมต่อ repository**  
   ![เลือก เชื่อมต่อ repository](/images/select-connect-repos.png)

2. เลือกหนึ่งใน repository ที่ติดตั้งไว้ที่คุณต้องการเชื่อมต่อ แล้วเลือก **บันทึก**  
   ![เลือก repository](/images/select-repo.png)

3. repository ที่เชื่อมต่อของคุณจะแสดงบนทั้งหน้าแรกและหน้ารายการ repository  
   ![repository ที่เชื่อมต่อแล้ว](/images/repo-connected.png)

---

## เริ่มการแปลอัตโนมัติ

1. เลือก repository ที่คุณเพิ่งเชื่อมต่อ  
   ![เลือก repository](/images/select-repo-to-detail.png)

2. ที่หน้ารายละเอียด repository เลือก **แก้ไข** ที่ด้านล่าง  
   ![เลือก แก้ไข](/images/select-edit.png)

3. ตั้งค่าการแปลของคุณ — สาขาเป้าหมาย (ค่าเริ่มต้น: `main`), ภาษาที่ต้องการแปล และภาษาต้นทาง (ค่าเริ่มต้น: `en`) จากนั้นเลือก **บันทึก**  
   ![ตั้งค่าการแปล](/images/configuration.png)

4. เลือก **เริ่ม & อัตโนมัติ**  
   Localizeflow จะเริ่มแปลเอกสารของคุณโดยอัตโนมัติและเปิด pull requests ทุกครั้งที่ต้นฉบับมีการเปลี่ยนแปลง  
   ![เริ่ม & อัตโนมัติ](/images/select-automate.png)
