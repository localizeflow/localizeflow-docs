Localizeflowは、ドキュメントを自動で翻訳し、ソースファイルが変更されるたびにプルリクエストを作成します。  
このガイドでは、GitHubアプリのインストール方法と、2分以内に最初の翻訳を実行する手順を説明します。

---

## サインインしてGitHubアプリをインストールする

1. **[localizeflow.com](https://localizeflow.com/)** にアクセスします。  
2. **無料トライアルを始める** を選択します。  
   ![無料トライアルを始めるを選択](/images/select-start-with-free-trial.png)  
3. **GitHubでサインイン** を選択します。  
   ![GitHubでサインイン](/images/select-sign-in-with-github.png)  
4. GitHubアカウントでサインインします。  
   ![GitHubログイン](/images/github-login.png)  
5. Localizeflow GitHubアプリをインストールしたいアカウント（個人アカウントまたは管理している組織）を選択します。  
   ![アカウントを選択](/images/select-which-account-to-use.png)  
6. Localizeflowにアクセスを許可したいリポジトリを選択し、**保存** をクリックします。  
   ![リポジトリを選択して保存](/images/select-repo-and-save.png)  
7. Localizeflowのホームページにリダイレクトされます。

> [!TIP]
> 後からリポジトリを追加したい場合は、ヘッダーのアカウントを選択し、**+ リポジトリを追加** を選んでください。  
> ![リポジトリを追加](/images/add-more-repo.png)

---

## リポジトリをLocalizeflowに接続する

1. Localizeflowのホームページで、**+ リポジトリを接続** を選択します。  
   ![リポジトリを接続を選択](/images/select-connect-repos.png)  
2. 接続したいインストール済みのリポジトリを選び、**保存** をクリックします。  
   ![リポジトリを選択](/images/select-repo.png)  
3. 接続されたリポジトリは、ホームページとリポジトリページの両方に表示されます。  
   ![接続されたリポジトリ](/images/repo-connected.png)  

---

## 自動翻訳を開始する

1. 先ほど接続したリポジトリを選択します。  
   ![リポジトリを選択](/images/select-repo-to-detail.png)  
2. リポジトリ詳細ページの下部にある **編集** を選択します。  
   ![編集を選択](/images/select-edit.png)  
3. 翻訳設定を行います — 対象ブランチ（デフォルト：`main`）、対象言語、ソース言語（デフォルト：`en`）を設定し、**保存** をクリックします。  
   ![翻訳設定を構成](/images/configuration.png)  
4. **開始＆自動化** を選択します。  
   Localizeflowがドキュメントを自動で翻訳し、ソースが変更されるたびにプルリクエストを作成します。  
   ![開始＆自動化](/images/select-automate.png)
