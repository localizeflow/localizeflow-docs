Localizeflow автоматично превежда вашата документация и отваря pull заявки всеки път, когато изходният файл се промени.  
Това ръководство ви показва как да инсталирате GitHub приложението и да стартирате първия си превод за по-малко от 2 минути.


> [!NOTE]
>
> В момента Localizeflow поддържа проекти за документация, базирани на GitHub  
> (например: AI for Beginners и повечето стандартни open-source хранилища).  
> 
> Поддръжката за модерни рамки за документация като Astro, Docusaurus и Hugo  
> е в активна разработка.


---

## Вход и инсталиране на GitHub приложението

1. Посетете **[localizeflow.com](https://localizeflow.com/)**.
2. Изберете **Start with free trial**.
   ![Select Start with free trial](/images/select-start-with-free-trial.png)
3. Изберете **Sign in with GitHub**.  
   ![Sign in with GitHub](/images/select-sign-in-with-github.png)
4. Влезте с вашия GitHub акаунт.  
   ![GitHub login](/images/github-login.png)
5. Изберете акаунта, в който искате да инсталирате Localizeflow GitHub приложението — вашият личен акаунт или организация, която управлявате.  
   ![Select account](/images/select-which-account-to-use.png)
6. Изберете хранилищата, до които искате Localizeflow да има достъп, след което изберете **Save**.  
   ![Select repo and save](/images/select-repo-and-save.png)
7. Ще бъдете пренасочени към началната страница на Localizeflow.

> [!TIP]
> За да добавите още хранилища по-късно, изберете акаунта си в заглавната част и изберете **+ Add more repositories**.  
> ![Add more repositories](/images/add-more-repo.png)

---

## Свържете вашите хранилища с Localizeflow

1. На началната страница на Localizeflow изберете **+ Connect repositories**.  
   ![Select connect repositories](/images/select-connect-repos.png)

2. Изберете едно от инсталираните хранилища, които искате да свържете, и изберете **Save**.  
   ![Select repository](/images/select-repo.png)

3. Вашите свързани хранилища вече ще се показват както на началната страница, така и на страницата с хранилища.  
   ![Connected repositories](/images/repo-connected.png)

---

## Стартирайте автоматичен превод

1. Изберете хранилището, което току-що свързахте.  
   ![Select repository](/images/select-repo-to-detail.png)

2. На страницата с детайли за хранилището изберете **Edit** в долната част.  
   ![Select edit](/images/select-edit.png)

3. Конфигурирайте настройките за превод — целевия клон (по подразбиране: `main`), целевите езици и изходния език (по подразбиране: `en`). Изберете **Save**.  
   ![Configure translation settings](/images/configuration.png)

4. Изберете **Start & Automate**.  
   Localizeflow сега ще превежда автоматично вашата документация и ще отваря pull заявки всеки път, когато изходният файл се промени.  
   ![Start & Automate](/images/select-automate.png)
