Localizeflow oversætter automatisk din dokumentation og åbner pull requests, når kildefilen ændres.  
Denne guide viser dig, hvordan du installerer GitHub App'en og kører din første oversættelse på under 2 minutter.


> [!NOTE]
>
> Localizeflow understøtter i øjeblikket dokumentationsprojekter baseret på GitHub  
> (for eksempel: AI for Beginners og de fleste standard open source repos).  
> 
> Understøttelse af moderne dokumentationsframeworks som Astro, Docusaurus og Hugo  
> er under aktiv udvikling.


---

## Log ind og installer GitHub App'en

1. Besøg **[localizeflow.com](https://localizeflow.com/)**.
2. Vælg **Start med gratis prøveperiode**.
   ![Select Start with free trial](/images/select-start-with-free-trial.png)
3. Vælg **Log ind med GitHub**.  
   ![Sign in with GitHub](/images/select-sign-in-with-github.png)
4. Log ind med din GitHub-konto.  
   ![GitHub login](/images/github-login.png)
5. Vælg den konto, hvor du vil installere Localizeflow GitHub App'en — din personlige konto eller en organisation, du administrerer.  
   ![Select account](/images/select-which-account-to-use.png)
6. Vælg de repositories, du ønsker, at Localizeflow skal have adgang til, og vælg derefter **Gem**.  
   ![Select repo and save](/images/select-repo-and-save.png)
7. Du bliver omdirigeret til Localizeflow startside.

> [!TIP]
> For at tilføje flere repositories senere, vælg din konto i headeren og vælg **+ Tilføj flere repositories**.  
> ![Add more repositories](/images/add-more-repo.png)

---

## Forbind dine repositories til Localizeflow

1. På Localizeflow startsiden, vælg **+ Forbind repositories**.  
   ![Select connect repositories](/images/select-connect-repos.png)

2. Vælg et af de installerede repositories, du vil forbinde, og vælg **Gem**.  
   ![Select repository](/images/select-repo.png)

3. Dine forbundne repositories vil nu vises både på startsiden og på repositories-siden.  
   ![Connected repositories](/images/repo-connected.png)

---

## Start automatisk oversættelse

1. Vælg det repository, du lige har forbundet.  
   ![Select repository](/images/select-repo-to-detail.png)

2. På repository-detaljesiden, vælg **Rediger** nederst.  
   ![Select edit](/images/select-edit.png)

3. Konfigurer dine oversættelsesindstillinger — målgren (standard: `main`), målsprog og kildesprog (standard: `en`). Vælg **Gem**.  
   ![Configure translation settings](/images/configuration.png)

4. Vælg **Start & Automatiser**.  
   Localizeflow vil nu automatisk oversætte din dokumentation og åbne pull requests, når kilden ændres.  
   ![Start & Automate](/images/select-automate.png)
