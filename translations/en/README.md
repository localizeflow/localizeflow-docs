Localizeflow automatically translates your documentation and creates pull requests whenever the source files are updated.  
This guide will show you how to install the GitHub App and run your first translation in under 2 minutes.

---

## Sign in and install the GitHub App

1. Go to **[localizeflow.com](https://localizeflow.com/)**.
2. Click **Start with free trial**.  
   ![Select Start with free trial](/images/select-start-with-free-trial.png)
3. Click **Sign in with GitHub**.  
   ![Sign in with GitHub](/images/select-sign-in-with-github.png)
4. Log in with your GitHub account.  
   ![GitHub login](/images/github-login.png)
5. Choose the account where you want to install the Localizeflow GitHub App — either your personal account or an organization you manage.  
   ![Select account](/images/select-which-account-to-use.png)
6. Select the repositories you want Localizeflow to access, then click **Save**.  
   ![Select repo and save](/images/select-repo-and-save.png)
7. You will be redirected to the Localizeflow homepage.

> [!TIP]
> To add more repositories later, click your account name in the header and select **+ Add more repositories**.  
> ![Add more repositories](/images/add-more-repo.png)

---

## Connect your repositories to Localizeflow

1. On the Localizeflow homepage, click **+ Connect repositories**.  
   ![Select connect repositories](/images/select-connect-repos.png)

2. Choose one of the installed repositories you want to connect and click **Save**.  
   ![Select repository](/images/select-repo.png)

3. Your connected repositories will now be visible on both the Home page and the Repositories page.  
   ![Connected repositories](/images/repo-connected.png)

---

## Start automatic translation

1. Select the repository you just connected.  
   ![Select repository](/images/select-repo-to-detail.png)

2. On the repository details page, click **Edit** at the bottom.  
   ![Select edit](/images/select-edit.png)

3. Set up your translation preferences — target branch (default: `main`), target languages, and source language (default: `en`). Click **Save**.  
   ![Configure translation settings](/images/configuration.png)

4. Click **Start & Automate**.  
   Localizeflow will now automatically translate your documentation and open pull requests whenever the source files change.  
   ![Start & Automate](/images/select-automate.png)
