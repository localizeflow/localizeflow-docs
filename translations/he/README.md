Localizeflow מתרגם אוטומטית את התיעוד שלך ופותח בקשות משיכה בכל פעם שקובץ המקור משתנה.  
המדריך הזה מראה לך איך להתקין את אפליקציית GitHub ולהריץ את התרגום הראשון שלך בפחות מ-2 דקות.

---

## התחבר והתקן את אפליקציית GitHub

1. בקר ב**[localizeflow.com](https://localizeflow.com/)**.
2. בחר **התחל עם ניסיון חינם**.  
   ![בחר התחלה עם ניסיון חינם](/images/select-start-with-free-trial.png)
3. בחר **התחבר עם GitHub**.  
   ![התחבר עם GitHub](/images/select-sign-in-with-github.png)
4. התחבר עם חשבון ה-GitHub שלך.  
   ![כניסה ל-GitHub](/images/github-login.png)
5. בחר את החשבון שבו תרצה להתקין את אפליקציית Localizeflow — חשבון אישי או ארגון שאתה מנהל.  
   ![בחר חשבון](/images/select-which-account-to-use.png)
6. בחר את המאגרי הקוד ש-Localizeflow יוכל לגשת אליהם, ואז לחץ על **שמור**.  
   ![בחר מאגר ושמור](/images/select-repo-and-save.png)
7. תועבר לדף הבית של Localizeflow.

> [!TIP]
> כדי להוסיף מאגרים נוספים מאוחר יותר, בחר את החשבון שלך בכותרת ובחר **+ הוסף מאגרים נוספים**.  
> ![הוסף מאגרים נוספים](/images/add-more-repo.png)

---

## חבר את המאגרי הקוד שלך ל-Localizeflow

1. בדף הבית של Localizeflow, בחר **+ חבר מאגרים**.  
   ![בחר חבר מאגרים](/images/select-connect-repos.png)

2. בחר אחד מהמאגרים שהתקנת שברצונך לחבר ולחץ על **שמור**.  
   ![בחר מאגר](/images/select-repo.png)

3. המאגר המחובר שלך יופיע כעת גם בדף הבית וגם בדף המאגרי קוד.  
   ![מאגרים מחוברים](/images/repo-connected.png)

---

## התחל תרגום אוטומטי

1. בחר את המאגר שהתחברת אליו זה עתה.  
   ![בחר מאגר](/images/select-repo-to-detail.png)

2. בדף פרטי המאגר, בחר **ערוך** בתחתית.  
   ![בחר ערוך](/images/select-edit.png)

3. הגדר את הגדרות התרגום שלך — ענף יעד (ברירת מחדל: `main`), שפות יעד, ושפת מקור (ברירת מחדל: `en`). לחץ על **שמור**.  
   ![הגדרות תרגום](/images/configuration.png)

4. בחר **התחל ואוטומציה**.  
   Localizeflow יתחיל לתרגם אוטומטית את התיעוד שלך ויפתח בקשות משיכה בכל פעם שהמקור משתנה.  
   ![התחל ואוטומציה](/images/select-automate.png)
