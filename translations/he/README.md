Localizeflow מתרגם אוטומטית את התיעוד שלך ופותח בקשות משיכה בכל פעם שקובץ המקור משתנה.  
מדריך זה מראה לך כיצד להתקין את אפליקציית GitHub ולהריץ את התרגום הראשון שלך בפחות מ-2 דקות.


> [!NOTE]
>
> Localizeflow תומך כרגע בפרויקטים של תיעוד מבוססי GitHub  
> (לדוגמה: AI for Beginners ורוב מאגרי הקוד הפתוחים הסטנדרטיים).  
> 
> התמיכה במסגרת תיעוד מודרניות כגון Astro, Docusaurus ו-Hugo  
> נמצאת בפיתוח פעיל.


---

## התחבר והתקן את אפליקציית GitHub

1. בקר ב**[localizeflow.com](https://localizeflow.com/)**.
2. בחר **Start with free trial**.
   ![Select Start with free trial](/images/select-start-with-free-trial.png)
3. בחר **Sign in with GitHub**.  
   ![Sign in with GitHub](/images/select-sign-in-with-github.png)
4. התחבר עם חשבון ה-GitHub שלך.  
   ![GitHub login](/images/github-login.png)
5. בחר את החשבון שבו ברצונך להתקין את אפליקציית Localizeflow ב-GitHub — חשבון אישי או ארגון שאתה מנהל.  
   ![Select account](/images/select-which-account-to-use.png)
6. בחר את המאגרי הקוד שברצונך ש-Localizeflow יקבל גישה אליהם, ואז בחר **Save**.  
   ![Select repo and save](/images/select-repo-and-save.png)
7. תועבר לדף הבית של Localizeflow.

> [!TIP]
> כדי להוסיף מאגרים נוספים מאוחר יותר, בחר את החשבון שלך בכותרת ובחר **+ Add more repositories**.  
> ![Add more repositories](/images/add-more-repo.png)

---

## חבר את מאגרי הקוד שלך ל-Localizeflow

1. בדף הבית של Localizeflow, בחר **+ Connect repositories**.  
   ![Select connect repositories](/images/select-connect-repos.png)

2. בחר אחד ממאגרי הקוד שהותקנו שברצונך לחבר ובחר **Save**.  
   ![Select repository](/images/select-repo.png)

3. מאגרי הקוד המחוברים שלך יופיעו כעת הן בדף הבית והן בדף המאגרים.  
   ![Connected repositories](/images/repo-connected.png)

---

## התחל תרגום אוטומטי

1. בחר את מאגר הקוד שחיברת זה עתה.  
   ![Select repository](/images/select-repo-to-detail.png)

2. בדף פרטי המאגר, בחר **Edit** בתחתית.  
   ![Select edit](/images/select-edit.png)

3. הגדר את הגדרות התרגום שלך — ענף יעד (ברירת מחדל: `main`), שפות יעד, ושפת מקור (ברירת מחדל: `en`). בחר **Save**.  
   ![Configure translation settings](/images/configuration.png)

4. בחר **Start & Automate**.  
   Localizeflow יתחיל כעת לתרגם אוטומטית את התיעוד שלך ויפתח בקשות משיכה בכל פעם שהמקור משתנה.  
   ![Start & Automate](/images/select-automate.png)
