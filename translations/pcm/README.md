Localizeflow go automatically translate your documentation and open pull requests anytime wey source file change.  
Dis guide go show you how to install the GitHub App and run your first translation for less than 2 minutes.

---

## Sign in and install the GitHub App

1. Go **[localizeflow.com](https://localizeflow.com/)**.
2. Choose **Start with free trial**.
   ![Select Start with free trial](/images/select-start-with-free-trial.png)
3. Choose **Sign in with GitHub**.  
   ![Sign in with GitHub](/images/select-sign-in-with-github.png)
4. Sign in with your GitHub account.  
   ![GitHub login](/images/github-login.png)
5. Pick the account wey you want install the Localizeflow GitHub App — your personal account or organization wey you dey manage.  
   ![Select account](/images/select-which-account-to-use.png)
6. Choose the repositories wey you want Localizeflow make e access, then click **Save**.  
   ![Select repo and save](/images/select-repo-and-save.png)
7. You go redirect go Localizeflow home page.

> [!TIP]
> If you want add more repositories later, just select your account for the header and choose **+ Add more repositories**.  
> ![Add more repositories](/images/add-more-repo.png)

---

## Connect your repositories to Localizeflow

1. For Localizeflow home page, select **+ Connect repositories**.  
   ![Select connect repositories](/images/select-connect-repos.png)

2. Choose one of the installed repositories wey you want connect and select **Save**.  
   ![Select repository](/images/select-repo.png)

3. Your connected repositories go show for both Home page and Repositories page.  
   ![Connected repositories](/images/repo-connected.png)

---

## Start automatic translation

1. Select the repository wey you just connect.  
   ![Select repository](/images/select-repo-to-detail.png)

2. For the repository detail page, select **Edit** for the bottom.  
   ![Select edit](/images/select-edit.png)

3. Set your translation settings — target branch (default: `main`), target languages, and source language (default: `en`). Then select **Save**.  
   ![Configure translation settings](/images/configuration.png)

4. Select **Start & Automate**.  
   Localizeflow go now automatically translate your documentation and open pull requests anytime source change.  
   ![Start & Automate](/images/select-automate.png)
