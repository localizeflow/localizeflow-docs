Localizeflow go automatically translate your documentation and open pull requests anytime source file change.  
Dis guide show you how to install di GitHub App and run your first translation under 2 minutes.


> [!NOTE]
>
> Localizeflow dey support GitHub-based documentation projects now  
> (example: AI for Beginners and most standard open-source repos).  
> 
> Support for beta documentation frameworks like Astro, Docusaurus, and Hugo  
> dey active for development.


---

## Sign in and install the GitHub App

1. Go **[localizeflow.com](https://localizeflow.com/)**.
2. Select **Start with free trial**.
   ![Select Start with free trial](../../../../translated images/pcm/select-start-with-free-trial.6c2e287133ff9c8b.webp)
3. Select **Sign in with GitHub**.  
   ![Sign in with GitHub](../../../../translated images/pcm/select-sign-in-with-github.f2850ffdd49cc894.webp)
4. Sign in with your GitHub account.  
   ![GitHub login](../../../../translated images/pcm/github-login.02bbaee0270b292e.webp)
5. Choose the account wey you wan install the Localizeflow GitHub App — your personal account or organization wey you dey manage.  
   ![Select account](../../../../translated images/pcm/select-which-account-to-use.7050f5ed0b773bb0.webp)
6. Select di repositories wey you want Localizeflow make e access, then choose **Save**.  
   ![Select repo and save](../../../../translated images/pcm/select-repo-and-save.5a95ae288aefec6e.webp)
7. You go redirect come Localizeflow home page.

> [!TIP]
> If you want add more repositories later, just select your account for di header and choose **+ Add more repositories**.  
> ![Add more repositories](../../../../translated images/pcm/add-more-repo.2ca5154473aaaafb.webp)

---

## Connect your repositories to Localizeflow

1. For Localizeflow home page, select **+ Connect repositories**.  
   ![Select connect repositories](../../../../translated images/pcm/select-connect-repos.8ac6f96f77dcc62c.webp)

2. Choose one of the installed repositories wey you want connect and select **Save**.  
   ![Select repository](../../../../translated images/pcm/select-repo.dce94db722b44cf3.webp)

3. Your connected repositories go show now for both Home page and Repositories page.  
   ![Connected repositories](../../../../translated images/pcm/repo-connected.9e5c21ee789fdcaa.webp)

---

## Start automatic translation

1. Select the repository wey you just connect.  
   ![Select repository](../../../../translated images/pcm/select-repo-to-detail.55e53233531f8518.webp)

2. For di repository detail page, select **Edit** for bottom.  
   ![Select edit](../../../../translated images/pcm/select-edit.225184c8c46d7001.webp)

3. Set up your translation settings — target branch (default: `main`), target languages, and source language (default: `en`). Then select **Save**.  
   ![Configure translation settings](../../../../translated images/pcm/configuration.ab55d0f8bab5711b.webp)

4. Select **Start & Automate**.  
   Localizeflow go now automatically translate your documentation and open pull requests any time di source change.  
   ![Start & Automate](../../../../translated images/pcm/select-automate.ff50c8d913b35a03.webp)
