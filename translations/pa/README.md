Localizeflow ਤੁਹਾਡੇ ਦਸਤਾਵੇਜ਼ਾਂ ਦਾ ਆਟੋਮੈਟਿਕ ਅਨੁਵਾਦ ਕਰਦਾ ਹੈ ਅਤੇ ਜਦੋਂ ਵੀ ਸਰੋਤ ਫਾਈਲ ਬਦਲਦੀ ਹੈ ਤਾਂ ਪੂਲ ਰਿਕਵੇਸਟ ਖੋਲ੍ਹਦਾ ਹੈ।  
ਇਹ ਗਾਈਡ ਤੁਹਾਨੂੰ ਦਿਖਾਉਂਦੀ ਹੈ ਕਿ ਕਿਵੇਂ GitHub ਐਪ ਇੰਸਟਾਲ ਕਰਨੀ ਹੈ ਅਤੇ 2 ਮਿੰਟਾਂ ਤੋਂ ਘੱਟ ਸਮੇਂ ਵਿੱਚ ਆਪਣਾ ਪਹਿਲਾ ਅਨੁਵਾਦ ਚਲਾਉਣਾ ਹੈ।


> [!NOTE]
>
> Localizeflow ਇਸ ਸਮੇਂ GitHub-ਅਧਾਰਿਤ ਦਸਤਾਵੇਜ਼ ਪ੍ਰੋਜੈਕਟਾਂ ਨੂੰ ਸਮਰਥਨ ਦਿੰਦਾ ਹੈ  
> (ਉਦਾਹਰਨ ਲਈ: AI for Beginners ਅਤੇ ਜ਼ਿਆਦਾਤਰ ਸਧਾਰਣ ਖੁੱਲ੍ਹੇ ਸਰੋਤ ਰਿਪੋਜ਼)।  
> 
> ਆਧੁਨਿਕ ਦਸਤਾਵੇਜ਼ ਫਰੇਮਵਰਕ ਜਿਵੇਂ ਕਿ Astro, Docusaurus, ਅਤੇ Hugo ਲਈ ਸਮਰਥਨ  
> ਸਰਗਰਮ ਵਿਕਾਸ ਵਿੱਚ ਹੈ।

## ਸਾਈਨ ਇਨ ਕਰੋ ਅਤੇ GitHub ਐਪ ਇੰਸਟਾਲ ਕਰੋ

1. ਜਾਓ **[localizeflow.com](https://localizeflow.com/)**।
2. ਚੁਣੋ **Start with free trial**।  
   ![Select Start with free trial](/images/select-start-with-free-trial.png)
3. ਚੁਣੋ **Sign in with GitHub**।  
   ![Sign in with GitHub](/images/select-sign-in-with-github.png)
4. ਆਪਣੇ GitHub ਖਾਤੇ ਨਾਲ ਸਾਈਨ ਇਨ ਕਰੋ।  
   ![GitHub login](/images/github-login.png)
5. ਉਹ ਖਾਤਾ ਚੁਣੋ ਜਿੱਥੇ ਤੁਸੀਂ Localizeflow GitHub ਐਪ ਇੰਸਟਾਲ ਕਰਨਾ ਚਾਹੁੰਦੇ ਹੋ — ਆਪਣਾ ਨਿੱਜੀ ਖਾਤਾ ਜਾਂ ਕੋਈ ਸੰਸਥਾ ਜਿਸਦਾ ਤੁਸੀਂ ਪ੍ਰਬੰਧਨ ਕਰਦੇ ਹੋ।  
   ![Select account](/images/select-which-account-to-use.png)
6. ਉਹ ਰਿਪੋਜ਼ ਚੁਣੋ ਜਿਨ੍ਹਾਂ ਤੱਕ Localizeflow ਦੀ ਪਹੁੰਚ ਚਾਹੁੰਦੇ ਹੋ, ਫਿਰ **Save** ਚੁਣੋ।  
   ![Select repo and save](/images/select-repo-and-save.png)
7. ਤੁਹਾਨੂੰ Localizeflow ਦੇ ਮੁੱਖ ਪੰਨੇ ਤੇ ਰੀਡਾਇਰੈਕਟ ਕੀਤਾ ਜਾਵੇਗਾ।

> [!TIP]
> ਬਾਅਦ ਵਿੱਚ ਹੋਰ ਰਿਪੋਜ਼ ਸ਼ਾਮਲ ਕਰਨ ਲਈ, ਹੈਡਰ ਵਿੱਚ ਆਪਣੇ ਖਾਤੇ ਨੂੰ ਚੁਣੋ ਅਤੇ **+ Add more repositories** ਚੁਣੋ।  
> ![Add more repositories](/images/add-more-repo.png)

---

## ਆਪਣੇ ਰਿਪੋਜ਼ ਨੂੰ Localizeflow ਨਾਲ ਜੋੜੋ

1. Localizeflow ਦੇ ਮੁੱਖ ਪੰਨੇ 'ਤੇ, **+ Connect repositories** ਚੁਣੋ।  
   ![Select connect repositories](/images/select-connect-repos.png)

2. ਇੰਸਟਾਲ ਕੀਤੇ ਗਏ ਰਿਪੋਜ਼ ਵਿੱਚੋਂ ਇੱਕ ਚੁਣੋ ਜਿਸਨੂੰ ਤੁਸੀਂ ਜੋੜਨਾ ਚਾਹੁੰਦੇ ਹੋ ਅਤੇ **Save** ਚੁਣੋ।  
   ![Select repository](/images/select-repo.png)

3. ਤੁਹਾਡੇ ਜੁੜੇ ਹੋਏ ਰਿਪੋਜ਼ ਹੁਣ ਮੁੱਖ ਪੰਨੇ ਅਤੇ ਰਿਪੋਜ਼ ਪੰਨੇ ਦੋਹਾਂ 'ਤੇ ਦਿਖਾਈ ਦੇਣਗੇ।  
   ![Connected repositories](/images/repo-connected.png)

---

## ਆਟੋਮੈਟਿਕ ਅਨੁਵਾਦ ਸ਼ੁਰੂ ਕਰੋ

1. ਉਹ ਰਿਪੋਜ਼ ਚੁਣੋ ਜੋ ਤੁਸੀਂ ਹੁਣੇ ਜੋੜਿਆ ਹੈ।  
   ![Select repository](/images/select-repo-to-detail.png)

2. ਰਿਪੋਜ਼ ਵੇਰਵਾ ਪੰਨੇ 'ਤੇ, ਹੇਠਾਂ **Edit** ਚੁਣੋ।  
   ![Select edit](/images/select-edit.png)

3. ਆਪਣੇ ਅਨੁਵਾਦ ਸੈਟਿੰਗਜ਼ ਸੰਰਚਿਤ ਕਰੋ — ਟਾਰਗੇਟ ਬ੍ਰਾਂਚ (ਡਿਫਾਲਟ: `main`), ਟਾਰਗੇਟ ਭਾਸ਼ਾਵਾਂ, ਅਤੇ ਸਰੋਤ ਭਾਸ਼ਾ (ਡਿਫਾਲਟ: `en`)। ਫਿਰ **Save** ਚੁਣੋ।  
   ![Configure translation settings](/images/configuration.png)

4. **Start & Automate** ਚੁਣੋ।  
   Localizeflow ਹੁਣ ਤੁਹਾਡੇ ਦਸਤਾਵੇਜ਼ਾਂ ਦਾ ਆਟੋਮੈਟਿਕ ਅਨੁਵਾਦ ਕਰੇਗਾ ਅਤੇ ਜਦੋਂ ਵੀ ਸਰੋਤ ਬਦਲੇਗਾ ਤਾਂ ਪੂਲ ਰਿਕਵੇਸਟ ਖੋਲ੍ਹੇਗਾ।  
   ![Start & Automate](/images/select-automate.png)
