Localizeflow உங்கள் ஆவணங்களை தானாக மொழிபெயர்க்கிறது மற்றும் மூலக் கோப்பு மாற்றப்படும் போதெல்லாம் புல் கோரிக்கைகளை திறக்கிறது.  
இந்த வழிகாட்டி GitHub செயலியை நிறுவி, 2 நிமிடங்களில் உங்கள் முதல் மொழிபெயர்ப்பை எப்படி இயக்குவது என்பதை காட்டுகிறது.

---

## உள்நுழைந்து GitHub செயலியை நிறுவுக

1. **[localizeflow.com](https://localizeflow.com/)** என்ற தளத்தை பார்வையிடவும்.
2. **Start with free trial** என்பதை தேர்ந்தெடுக்கவும்.  
   ![Select Start with free trial](/images/select-start-with-free-trial.png)
3. **Sign in with GitHub** என்பதை தேர்ந்தெடுக்கவும்.  
   ![Sign in with GitHub](/images/select-sign-in-with-github.png)
4. உங்கள் GitHub கணக்கில் உள்நுழைக.  
   ![GitHub login](/images/github-login.png)
5. Localizeflow GitHub செயலியை நிறுவ விரும்பும் கணக்கை தேர்ந்தெடுக்கவும் — உங்கள் தனிப்பட்ட கணக்கு அல்லது நீங்கள் நிர்வகிக்கும் அமைப்பு.  
   ![Select account](/images/select-which-account-to-use.png)
6. Localizeflow அணுக விரும்பும் ரெப்போசிடரிகளை தேர்ந்தெடுத்து, **Save** என்பதை அழுத்தவும்.  
   ![Select repo and save](/images/select-repo-and-save.png)
7. நீங்கள் Localizeflow முகப்புப் பக்கத்திற்கு திருப்பி அனுப்பப்படுவீர்கள்.

> [!TIP]
> பின்னர் மேலும் ரெப்போசிடரிகளை சேர்க்க, தலைப்பில் உங்கள் கணக்கை தேர்ந்தெடுத்து **+ Add more repositories** என்பதை கிளிக் செய்யவும்.  
> ![Add more repositories](/images/add-more-repo.png)

---

## உங்கள் ரெப்போசிடரிகளை Localizeflow-க்கு இணைக்கவும்

1. Localizeflow முகப்புப் பக்கத்தில் **+ Connect repositories** என்பதை தேர்ந்தெடுக்கவும்.  
   ![Select connect repositories](/images/select-connect-repos.png)

2. நீங்கள் இணைக்க விரும்பும் நிறுவப்பட்ட ரெப்போசிடரிகளில் ஒன்றை தேர்ந்தெடுத்து **Save** என்பதை அழுத்தவும்.  
   ![Select repository](/images/select-repo.png)

3. உங்கள் இணைக்கப்பட்ட ரெப்போசிடரிகள் இப்போது முகப்புப் பக்கத்திலும் ரெப்போசிடரிகள் பக்கத்திலும் தோன்றும்.  
   ![Connected repositories](/images/repo-connected.png)

---

## தானாக மொழிபெயர்ப்பு தொடங்கவும்

1. நீங்கள் இப்போது இணைத்த ரெப்போசிடரியை தேர்ந்தெடுக்கவும்.  
   ![Select repository](/images/select-repo-to-detail.png)

2. ரெப்போசிடரி விவரப் பக்கத்தில், கீழே உள்ள **Edit** என்பதை தேர்ந்தெடுக்கவும்.  
   ![Select edit](/images/select-edit.png)

3. உங்கள் மொழிபெயர்ப்பு அமைப்புகளை அமைக்கவும் — இலக்கு கிளை (இயல்புநிலை: `main`), இலக்கு மொழிகள் மற்றும் மூல மொழி (இயல்புநிலை: `en`). **Save** என்பதை அழுத்தவும்.  
   ![Configure translation settings](/images/configuration.png)

4. **Start & Automate** என்பதை தேர்ந்தெடுக்கவும்.  
   Localizeflow இப்போது உங்கள் ஆவணங்களை தானாக மொழிபெயர்த்து, மூலத்தில் மாற்றம் ஏற்பட்டால் புல் கோரிக்கைகளை திறக்கும்.  
   ![Start & Automate](/images/select-automate.png)
