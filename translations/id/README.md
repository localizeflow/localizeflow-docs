Localizeflow secara otomatis menerjemahkan dokumentasi Anda dan membuka pull request setiap kali file sumber berubah.  
Panduan ini menunjukkan cara menginstal Aplikasi GitHub dan menjalankan terjemahan pertama Anda dalam waktu kurang dari 2 menit.

---

## Masuk dan pasang Aplikasi GitHub

1. Kunjungi **[localizeflow.com](https://localizeflow.com/)**.
2. Pilih **Mulai dengan uji coba gratis**.  
   ![Pilih Mulai dengan uji coba gratis](/images/select-start-with-free-trial.png)
3. Pilih **Masuk dengan GitHub**.  
   ![Masuk dengan GitHub](/images/select-sign-in-with-github.png)
4. Masuk menggunakan akun GitHub Anda.  
   ![Login GitHub](/images/github-login.png)
5. Pilih akun tempat Anda ingin memasang Aplikasi GitHub Localizeflow — akun pribadi Anda atau organisasi yang Anda kelola.  
   ![Pilih akun](/images/select-which-account-to-use.png)
6. Pilih repositori yang ingin diakses oleh Localizeflow, lalu pilih **Simpan**.  
   ![Pilih repo dan simpan](/images/select-repo-and-save.png)
7. Anda akan diarahkan ke halaman utama Localizeflow.

> [!TIP]
> Untuk menambahkan repositori lebih banyak nanti, pilih akun Anda di header dan pilih **+ Tambah repositori lain**.  
> ![Tambah repositori lain](/images/add-more-repo.png)

---

## Hubungkan repositori Anda ke Localizeflow

1. Di halaman utama Localizeflow, pilih **+ Hubungkan repositori**.  
   ![Pilih hubungkan repositori](/images/select-connect-repos.png)

2. Pilih salah satu repositori yang sudah terpasang yang ingin Anda hubungkan, lalu pilih **Simpan**.  
   ![Pilih repositori](/images/select-repo.png)

3. Repositori yang terhubung sekarang akan muncul di halaman Beranda dan halaman Repositori.  
   ![Repositori terhubung](/images/repo-connected.png)

---

## Mulai terjemahan otomatis

1. Pilih repositori yang baru saja Anda hubungkan.  
   ![Pilih repositori](/images/select-repo-to-detail.png)

2. Di halaman detail repositori, pilih **Edit** di bagian bawah.  
   ![Pilih edit](/images/select-edit.png)

3. Atur pengaturan terjemahan Anda — cabang target (default: `main`), bahasa target, dan bahasa sumber (default: `en`). Pilih **Simpan**.  
   ![Atur pengaturan terjemahan](/images/configuration.png)

4. Pilih **Mulai & Otomatiskan**.  
   Localizeflow sekarang akan secara otomatis menerjemahkan dokumentasi Anda dan membuka pull request setiap kali sumber berubah.  
   ![Mulai & Otomatiskan](/images/select-automate.png)
