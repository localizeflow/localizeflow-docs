Localizeflow secara otomatis menerjemahkan dokumentasi Anda dan membuka pull request setiap kali file sumber berubah.  
Panduan ini menunjukkan cara menginstal GitHub App dan menjalankan terjemahan pertama Anda dalam waktu kurang dari 2 menit.


> [!NOTE]
>
> Localizeflow saat ini mendukung proyek dokumentasi berbasis GitHub  
> (misalnya: AI for Beginners dan sebagian besar repositori open-source standar).  
> 
> Dukungan untuk kerangka dokumentasi modern seperti Astro, Docusaurus, dan Hugo  
> sedang dalam pengembangan aktif.


---

## Masuk dan instal GitHub App

1. Kunjungi **[localizeflow.com](https://localizeflow.com/)**.
2. Pilih **Mulai dengan uji coba gratis**.
   ![Select Start with free trial](/images/select-start-with-free-trial.png)
3. Pilih **Masuk dengan GitHub**.  
   ![Sign in with GitHub](/images/select-sign-in-with-github.png)
4. Masuk dengan akun GitHub Anda.  
   ![GitHub login](/images/github-login.png)
5. Pilih akun tempat Anda ingin menginstal Localizeflow GitHub App — akun pribadi Anda atau organisasi yang Anda kelola.  
   ![Select account](/images/select-which-account-to-use.png)
6. Pilih repositori yang ingin Anda beri akses ke Localizeflow, lalu pilih **Simpan**.  
   ![Select repo and save](/images/select-repo-and-save.png)
7. Anda akan diarahkan ke halaman utama Localizeflow.

> [!TIP]
> Untuk menambahkan lebih banyak repositori nanti, pilih akun Anda di header dan pilih **+ Tambah lebih banyak repositori**.  
> ![Add more repositories](/images/add-more-repo.png)

---

## Hubungkan repositori Anda ke Localizeflow

1. Di halaman utama Localizeflow, pilih **+ Hubungkan repositori**.  
   ![Select connect repositories](/images/select-connect-repos.png)

2. Pilih salah satu repositori yang sudah diinstal yang ingin Anda hubungkan dan pilih **Simpan**.  
   ![Select repository](/images/select-repo.png)

3. Repositori yang terhubung sekarang akan muncul di halaman Beranda dan halaman Repositori.  
   ![Connected repositories](/images/repo-connected.png)

---

## Mulai terjemahan otomatis

1. Pilih repositori yang baru saja Anda hubungkan.  
   ![Select repository](/images/select-repo-to-detail.png)

2. Di halaman detail repositori, pilih **Edit** di bagian bawah.  
   ![Select edit](/images/select-edit.png)

3. Konfigurasikan pengaturan terjemahan Anda — cabang target (default: `main`), bahasa target, dan bahasa sumber (default: `en`). Pilih **Simpan**.  
   ![Configure translation settings](/images/configuration.png)

4. Pilih **Mulai & Otomatiskan**.  
   Localizeflow sekarang akan secara otomatis menerjemahkan dokumentasi Anda dan membuka pull request setiap kali sumber berubah.  
   ![Start & Automate](/images/select-automate.png)
