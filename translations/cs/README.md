Localizeflow automaticky překládá vaši dokumentaci a otevírá pull requesty vždy, když se změní zdrojový soubor.  
Tento průvodce vám ukáže, jak nainstalovat GitHub App a spustit první překlad během méně než 2 minut.

---

## Přihlaste se a nainstalujte GitHub App

1. Navštivte **[localizeflow.com](https://localizeflow.com/)**.
2. Vyberte **Start with free trial**.  
   ![Vyberte Start with free trial](/images/select-start-with-free-trial.png)
3. Zvolte **Sign in with GitHub**.  
   ![Přihlaste se přes GitHub](/images/select-sign-in-with-github.png)
4. Přihlaste se pomocí svého GitHub účtu.  
   ![Přihlášení do GitHub](/images/github-login.png)
5. Vyberte účet, do kterého chcete nainstalovat Localizeflow GitHub App — svůj osobní účet nebo organizaci, kterou spravujete.  
   ![Vyberte účet](/images/select-which-account-to-use.png)
6. Vyberte repozitáře, ke kterým chcete, aby měl Localizeflow přístup, a poté klikněte na **Save**.  
   ![Vyberte repozitář a uložte](/images/select-repo-and-save.png)
7. Budete přesměrováni na domovskou stránku Localizeflow.

> [!TIP]
> Pokud chcete později přidat další repozitáře, klikněte na svůj účet v záhlaví a vyberte **+ Add more repositories**.  
> ![Přidat další repozitáře](/images/add-more-repo.png)

---

## Připojte své repozitáře k Localizeflow

1. Na domovské stránce Localizeflow klikněte na **+ Connect repositories**.  
   ![Vyberte připojení repozitářů](/images/select-connect-repos.png)

2. Vyberte jeden z nainstalovaných repozitářů, který chcete připojit, a klikněte na **Save**.  
   ![Vyberte repozitář](/images/select-repo.png)

3. Připojené repozitáře se nyní zobrazí jak na domovské stránce, tak na stránce Repositories.  
   ![Připojené repozitáře](/images/repo-connected.png)

---

## Spusťte automatický překlad

1. Vyberte repozitář, který jste právě připojili.  
   ![Vyberte repozitář](/images/select-repo-to-detail.png)

2. Na stránce s detaily repozitáře klikněte dole na **Edit**.  
   ![Vyberte upravit](/images/select-edit.png)

3. Nastavte překladové parametry — cílovou větev (výchozí: `main`), cílové jazyky a zdrojový jazyk (výchozí: `en`). Klikněte na **Save**.  
   ![Nastavení překladu](/images/configuration.png)

4. Klikněte na **Start & Automate**.  
   Localizeflow nyní automaticky přeloží vaši dokumentaci a otevře pull requesty vždy, když se změní zdroj.  
   ![Start & Automate](/images/select-automate.png)
