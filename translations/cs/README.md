Localizeflow automaticky překládá vaši dokumentaci a otevírá pull requesty vždy, když se změní zdrojový soubor.  
Tento průvodce vám ukáže, jak nainstalovat GitHub App a spustit první překlad za méně než 2 minuty.


> [!NOTE]
>
> Localizeflow aktuálně podporuje dokumentační projekty založené na GitHubu  
> (například: AI for Beginners a většina standardních open-source repozitářů).  
> 
> Podpora moderních dokumentačních frameworků jako Astro, Docusaurus a Hugo  
> je v aktivním vývoji.


---

## Přihlaste se a nainstalujte GitHub App

1. Navštivte **[localizeflow.com](https://localizeflow.com/)**.
2. Vyberte **Start with free trial**.
   ![Select Start with free trial](/images/select-start-with-free-trial.png)
3. Vyberte **Sign in with GitHub**.  
   ![Sign in with GitHub](/images/select-sign-in-with-github.png)
4. Přihlaste se pomocí svého GitHub účtu.  
   ![GitHub login](/images/github-login.png)
5. Zvolte účet, do kterého chcete nainstalovat Localizeflow GitHub App — svůj osobní účet nebo organizaci, kterou spravujete.  
   ![Select account](/images/select-which-account-to-use.png)
6. Vyberte repozitáře, ke kterým chcete, aby měl Localizeflow přístup, a poté zvolte **Save**.  
   ![Select repo and save](/images/select-repo-and-save.png)
7. Budete přesměrováni na domovskou stránku Localizeflow.

> [!TIP]
> Pro přidání dalších repozitářů později vyberte svůj účet v záhlaví a zvolte **+ Add more repositories**.  
> ![Add more repositories](/images/add-more-repo.png)

---

## Připojte své repozitáře k Localizeflow

1. Na domovské stránce Localizeflow vyberte **+ Connect repositories**.  
   ![Select connect repositories](/images/select-connect-repos.png)

2. Vyberte jeden z nainstalovaných repozitářů, který chcete připojit, a zvolte **Save**.  
   ![Select repository](/images/select-repo.png)

3. Vaše připojené repozitáře se nyní zobrazí jak na domovské stránce, tak na stránce Repositories.  
   ![Connected repositories](/images/repo-connected.png)

---

## Spusťte automatický překlad

1. Vyberte repozitář, který jste právě připojili.  
   ![Select repository](/images/select-repo-to-detail.png)

2. Na stránce detailu repozitáře vyberte dole **Edit**.  
   ![Select edit](/images/select-edit.png)

3. Nakonfigurujte nastavení překladu — cílovou větev (výchozí: `main`), cílové jazyky a zdrojový jazyk (výchozí: `en`). Vyberte **Save**.  
   ![Configure translation settings](/images/configuration.png)

4. Vyberte **Start & Automate**.  
   Localizeflow nyní automaticky přeloží vaši dokumentaci a otevře pull requesty vždy, když se zdroj změní.  
   ![Start & Automate](/images/select-automate.png)
