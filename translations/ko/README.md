Localizeflow는 문서가 변경될 때마다 자동으로 번역하고 풀 리퀘스트를 생성합니다.  
이 가이드는 GitHub 앱 설치부터 2분 이내에 첫 번역을 실행하는 방법을 안내합니다.

---

## GitHub 앱 로그인 및 설치

1. **[localizeflow.com](https://localizeflow.com/)**에 접속하세요.  
2. **무료 체험 시작(Start with free trial)**을 선택하세요.  
   ![무료 체험 시작 선택](/images/select-start-with-free-trial.png)  
3. **GitHub로 로그인(Sign in with GitHub)**을 선택하세요.  
   ![GitHub로 로그인](/images/select-sign-in-with-github.png)  
4. GitHub 계정으로 로그인하세요.  
   ![GitHub 로그인](/images/github-login.png)  
5. Localizeflow GitHub 앱을 설치할 계정을 선택하세요 — 개인 계정 또는 관리하는 조직 중에서 선택할 수 있습니다.  
   ![계정 선택](/images/select-which-account-to-use.png)  
6. Localizeflow가 접근할 저장소를 선택한 후 **저장(Save)**을 클릭하세요.  
   ![저장소 선택 및 저장](/images/select-repo-and-save.png)  
7. Localizeflow 홈 페이지로 리디렉션됩니다.

> [!TIP]
> 나중에 저장소를 추가하려면 헤더에서 계정을 선택한 후 **+ 저장소 추가(Add more repositories)**를 클릭하세요.  
> ![저장소 추가](/images/add-more-repo.png)

---

## 저장소를 Localizeflow에 연결하기

1. Localizeflow 홈 페이지에서 **+ 저장소 연결(Connect repositories)**을 선택하세요.  
   ![저장소 연결 선택](/images/select-connect-repos.png)

2. 연결할 설치된 저장소 중 하나를 선택하고 **저장(Save)**을 클릭하세요.  
   ![저장소 선택](/images/select-repo.png)

3. 연결된 저장소가 홈 페이지와 저장소 페이지 모두에 표시됩니다.  
   ![연결된 저장소](/images/repo-connected.png)

---

## 자동 번역 시작하기

1. 방금 연결한 저장소를 선택하세요.  
   ![저장소 선택](/images/select-repo-to-detail.png)

2. 저장소 상세 페이지 하단에서 **편집(Edit)**을 선택하세요.  
   ![편집 선택](/images/select-edit.png)

3. 번역 설정을 구성하세요 — 대상 브랜치(기본값: `main`), 대상 언어, 원본 언어(기본값: `en`). 설정 후 **저장(Save)**을 클릭하세요.  
   ![번역 설정 구성](/images/configuration.png)

4. **시작 및 자동화(Start & Automate)**를 선택하세요.  
   Localizeflow가 이제 문서 변경 시 자동으로 번역하고 풀 리퀘스트를 생성합니다.  
   ![시작 및 자동화](/images/select-automate.png)
