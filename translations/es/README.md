Localizeflow traduce automáticamente tu documentación y abre solicitudes de extracción cada vez que el archivo fuente cambia.  
Esta guía te muestra cómo instalar la aplicación de GitHub y realizar tu primera traducción en menos de 2 minutos.

---

## Inicia sesión e instala la aplicación de GitHub

1. Visita **[localizeflow.com](https://localizeflow.com/)**.
2. Selecciona **Comenzar con prueba gratuita**.  
   ![Selecciona Comenzar con prueba gratuita](/images/select-start-with-free-trial.png)
3. Selecciona **Iniciar sesión con GitHub**.  
   ![Iniciar sesión con GitHub](/images/select-sign-in-with-github.png)
4. Inicia sesión con tu cuenta de GitHub.  
   ![Inicio de sesión en GitHub](/images/github-login.png)
5. Elige la cuenta donde quieres instalar la aplicación Localizeflow de GitHub — tu cuenta personal o una organización que administres.  
   ![Selecciona cuenta](/images/select-which-account-to-use.png)
6. Selecciona los repositorios a los que quieres que Localizeflow tenga acceso y luego elige **Guardar**.  
   ![Selecciona repositorio y guarda](/images/select-repo-and-save.png)
7. Serás redirigido a la página principal de Localizeflow.

> [!TIP]
> Para agregar más repositorios después, selecciona tu cuenta en el encabezado y elige **+ Agregar más repositorios**.  
> ![Agregar más repositorios](/images/add-more-repo.png)

---

## Conecta tus repositorios a Localizeflow

1. En la página principal de Localizeflow, selecciona **+ Conectar repositorios**.  
   ![Selecciona conectar repositorios](/images/select-connect-repos.png)

2. Elige uno de los repositorios instalados que quieres conectar y selecciona **Guardar**.  
   ![Selecciona repositorio](/images/select-repo.png)

3. Tus repositorios conectados aparecerán ahora tanto en la página principal como en la página de Repositorios.  
   ![Repositorios conectados](/images/repo-connected.png)

---

## Comienza la traducción automática

1. Selecciona el repositorio que acabas de conectar.  
   ![Selecciona repositorio](/images/select-repo-to-detail.png)

2. En la página de detalles del repositorio, selecciona **Editar** en la parte inferior.  
   ![Selecciona editar](/images/select-edit.png)

3. Configura tus ajustes de traducción — rama destino (por defecto: `main`), idiomas destino y idioma fuente (por defecto: `en`). Selecciona **Guardar**.  
   ![Configura ajustes de traducción](/images/configuration.png)

4. Selecciona **Iniciar y Automatizar**.  
   Localizeflow ahora traducirá automáticamente tu documentación y abrirá solicitudes de extracción cada vez que la fuente cambie.  
   ![Iniciar y Automatizar](/images/select-automate.png)
