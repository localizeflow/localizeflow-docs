> **¿Prefieres Clonar Localmente?**

> Este repositorio incluye más de 50 traducciones de idiomas que aumentan significativamente el tamaño de la descarga. Para clonar sin traducciones, usa sparse checkout:
> ```bash
> git clone --filter=blob:none --sparse https://github.com/localizeflow/localizeflow-docs.git
> cd localizeflow-docs
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
> Esto te ofrece todo lo que necesitas para completar el curso con una descarga mucho más rápida.

Localizeflow traduce automáticamente tu documentación y abre solicitudes de extracción cada vez que cambia el archivo fuente.  
Esta guía te muestra cómo instalar la App de GitHub y ejecutar tu primera traducción en menos de 2 minutos.


> [!NOTE]
>
> Localizeflow actualmente soporta proyectos de documentación basados en GitHub
> (por ejemplo: AI for Beginners y la mayoría de los repositorios open-source estándar).  
> 
> El soporte para frameworks modernos de documentación como Astro, Docusaurus y Hugo  
> está en desarrollo activo.


---

## Iniciar sesión e instalar la App de GitHub

1. Visita **[localizeflow.com](https://localizeflow.com/)**.
2. Selecciona **Comenzar con prueba gratuita**.
   ![Selecciona Comenzar con prueba gratuita](../../../../translated images/es/select-start-with-free-trial.6c2e287133ff9c8b.webp)
3. Selecciona **Iniciar sesión con GitHub**.  
   ![Iniciar sesión con GitHub](../../../../translated images/es/select-sign-in-with-github.f2850ffdd49cc894.webp)
4. Inicia sesión con tu cuenta de GitHub.  
   ![Inicio de sesión en GitHub](../../../../translated images/es/github-login.02bbaee0270b292e.webp)
5. Elige la cuenta donde quieres instalar la App de GitHub de Localizeflow — tu cuenta personal o una organización que administres.  
   ![Seleccionar cuenta](../../../../translated images/es/select-which-account-to-use.7050f5ed0b773bb0.webp)
6. Selecciona los repositorios a los que quieres que Localizeflow acceda, luego elige **Guardar**.  
   ![Seleccionar repositorio y guardar](../../../../translated images/es/select-repo-and-save.5a95ae288aefec6e.webp)
7. Serás redirigido a la página principal de Localizeflow.

> [!TIP]
> Para agregar más repositorios después, selecciona tu cuenta en el encabezado y elige **+ Agregar más repositorios**.  
> ![Agregar más repositorios](../../../../translated images/es/add-more-repo.2ca5154473aaaafb.webp)

---

## Conecta tus repositorios a Localizeflow

1. En la página principal de Localizeflow, selecciona **+ Conectar repositorios**.  
   ![Seleccionar conectar repositorios](../../../../translated images/es/select-connect-repos.8ac6f96f77dcc62c.webp)

2. Elige uno de los repositorios instalados que deseas conectar y selecciona **Guardar**.  
   ![Seleccionar repositorio](../../../../translated images/es/select-repo.dce94db722b44cf3.webp)

3. Tus repositorios conectados aparecerán ahora tanto en la página principal como en la página de Repositorios.  
   ![Repositorios conectados](../../../../translated images/es/repo-connected.9e5c21ee789fdcaa.webp)

---

## Comienza la traducción automática

1. Selecciona el repositorio que acabas de conectar.  
   ![Seleccionar repositorio](../../../../translated images/es/select-repo-to-detail.55e53233531f8518.webp)

2. En la página de detalles del repositorio, selecciona **Editar** en la parte inferior.  
   ![Seleccionar editar](../../../../translated images/es/select-edit.225184c8c46d7001.webp)

3. Configura los ajustes de traducción — rama destino (por defecto: `main`), idiomas destino y idioma fuente (por defecto: `en`). Selecciona **Guardar**.  
   ![Configurar ajustes de traducción](../../../../translated images/es/configuration.ab55d0f8bab5711b.webp)

4. Selecciona **Iniciar y Automatizar**.  
   Localizeflow ahora traducirá automáticamente tu documentación y abrirá solicitudes de extracción cada vez que la fuente cambie.  
   ![Iniciar y Automatizar](../../../../translated images/es/select-automate.ff50c8d913b35a03.webp)
