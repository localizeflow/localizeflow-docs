Localizeflow traduce automáticamente tu documentación y abre solicitudes de extracción cada vez que el archivo fuente cambia.  
Esta guía te muestra cómo instalar la aplicación de GitHub y ejecutar tu primera traducción en menos de 2 minutos.


> [!NOTE]
>
> Actualmente, Localizeflow soporta proyectos de documentación basados en GitHub  
> (por ejemplo: AI for Beginners y la mayoría de repositorios estándar de código abierto).  
> 
> El soporte para frameworks modernos de documentación como Astro, Docusaurus y Hugo  
> está en desarrollo activo.


---

## Inicia sesión e instala la aplicación de GitHub

1. Visita **[localizeflow.com](https://localizeflow.com/)**.
2. Selecciona **Comenzar con prueba gratuita**.
   ![Select Start with free trial](/images/select-start-with-free-trial.png)
3. Selecciona **Iniciar sesión con GitHub**.  
   ![Sign in with GitHub](/images/select-sign-in-with-github.png)
4. Inicia sesión con tu cuenta de GitHub.  
   ![GitHub login](/images/github-login.png)
5. Elige la cuenta donde quieres instalar la aplicación de GitHub de Localizeflow — tu cuenta personal o una organización que administres.  
   ![Select account](/images/select-which-account-to-use.png)
6. Selecciona los repositorios a los que quieres que Localizeflow tenga acceso, luego elige **Guardar**.  
   ![Select repo and save](/images/select-repo-and-save.png)
7. Serás redirigido a la página principal de Localizeflow.

> [!TIP]
> Para agregar más repositorios después, selecciona tu cuenta en el encabezado y elige **+ Agregar más repositorios**.  
> ![Add more repositories](/images/add-more-repo.png)

---

## Conecta tus repositorios a Localizeflow

1. En la página principal de Localizeflow, selecciona **+ Conectar repositorios**.  
   ![Select connect repositories](/images/select-connect-repos.png)

2. Elige uno de los repositorios instalados que quieres conectar y selecciona **Guardar**.  
   ![Select repository](/images/select-repo.png)

3. Tus repositorios conectados aparecerán ahora tanto en la página principal como en la página de Repositorios.  
   ![Connected repositories](/images/repo-connected.png)

---

## Comienza la traducción automática

1. Selecciona el repositorio que acabas de conectar.  
   ![Select repository](/images/select-repo-to-detail.png)

2. En la página de detalles del repositorio, selecciona **Editar** en la parte inferior.  
   ![Select edit](/images/select-edit.png)

3. Configura tus ajustes de traducción — rama objetivo (por defecto: `main`), idiomas objetivo y idioma fuente (por defecto: `en`). Selecciona **Guardar**.  
   ![Configure translation settings](/images/configuration.png)

4. Selecciona **Iniciar y Automatizar**.  
   Localizeflow ahora traducirá automáticamente tu documentación y abrirá solicitudes de extracción cada vez que la fuente cambie.  
   ![Start & Automate](/images/select-automate.png)
