يقوم Localizeflow بترجمة توثيقاتك تلقائيًا ويفتح طلبات سحب كلما تغير ملف المصدر.  
يوضح لك هذا الدليل كيفية تثبيت تطبيق GitHub وتشغيل أول ترجمة لك في أقل من دقيقتين.


> [!NOTE]
>
> يدعم Localizeflow حاليًا مشاريع التوثيق المعتمدة على GitHub  
> (على سبيل المثال: AI for Beginners ومعظم مستودعات المصدر المفتوح القياسية).  
> 
> الدعم لأُطُر التوثيق الحديثة مثل Astro وDocusaurus وHugo  
> قيد التطوير النشط.


---

## تسجيل الدخول وتثبيت تطبيق GitHub

1. قم بزيارة **[localizeflow.com](https://localizeflow.com/)**.
2. اختر **ابدأ مع النسخة التجريبية المجانية**.
   ![Select Start with free trial](/images/select-start-with-free-trial.png)
3. اختر **تسجيل الدخول باستخدام GitHub**.  
   ![Sign in with GitHub](/images/select-sign-in-with-github.png)
4. سجّل الدخول باستخدام حساب GitHub الخاص بك.  
   ![GitHub login](/images/github-login.png)
5. اختر الحساب الذي تريد تثبيت تطبيق Localizeflow GitHub عليه — حسابك الشخصي أو منظمة تديرها.  
   ![Select account](/images/select-which-account-to-use.png)
6. اختر المستودعات التي تريد أن يصل إليها Localizeflow، ثم اختر **حفظ**.  
   ![Select repo and save](/images/select-repo-and-save.png)
7. سيتم توجيهك إلى الصفحة الرئيسية لـ Localizeflow.

> [!TIP]
> لإضافة المزيد من المستودعات لاحقًا، اختر حسابك في الرأس ثم اختر **+ إضافة المزيد من المستودعات**.  
> ![Add more repositories](/images/add-more-repo.png)

---

## ربط مستودعاتك بـ Localizeflow

1. في الصفحة الرئيسية لـ Localizeflow، اختر **+ ربط المستودعات**.  
   ![Select connect repositories](/images/select-connect-repos.png)

2. اختر أحد المستودعات المثبتة التي تريد ربطها ثم اختر **حفظ**.  
   ![Select repository](/images/select-repo.png)

3. ستظهر مستودعاتك المرتبطة الآن في كل من الصفحة الرئيسية وصفحة المستودعات.  
   ![Connected repositories](/images/repo-connected.png)

---

## بدء الترجمة التلقائية

1. اختر المستودع الذي قمت بربطه للتو.  
   ![Select repository](/images/select-repo-to-detail.png)

2. في صفحة تفاصيل المستودع، اختر **تحرير** في الأسفل.  
   ![Select edit](/images/select-edit.png)

3. قم بتكوين إعدادات الترجمة — الفرع المستهدف (الافتراضي: `main`)، اللغات المستهدفة، ولغة المصدر (الافتراضي: `en`). اختر **حفظ**.  
   ![Configure translation settings](/images/configuration.png)

4. اختر **ابدأ وفعّل الأتمتة**.  
   سيقوم Localizeflow الآن بترجمة توثيقاتك تلقائيًا وفتح طلبات السحب كلما تغير المصدر.  
   ![Start & Automate](/images/select-automate.png)
