Localizeflow, dokümantasyonunuzu otomatik olarak çevirir ve kaynak dosya değiştiğinde pull request açar.  
Bu kılavuz, GitHub Uygulamasını nasıl kuracağınızı ve ilk çevirinizi 2 dakikadan kısa sürede nasıl yapacağınızı gösterir.

---

## GitHub Uygulamasına giriş yapın ve kurun

1. **[localizeflow.com](https://localizeflow.com/)** adresini ziyaret edin.
2. **Ücretsiz deneme ile başla** seçeneğini tıklayın.  
   ![Ücretsiz deneme ile başla seçin](/images/select-start-with-free-trial.png)
3. **GitHub ile giriş yap** seçeneğini seçin.  
   ![GitHub ile giriş yap](/images/select-sign-in-with-github.png)
4. GitHub hesabınızla giriş yapın.  
   ![GitHub giriş ekranı](/images/github-login.png)
5. Localizeflow GitHub Uygulamasını kurmak istediğiniz hesabı seçin — kişisel hesabınız veya yönettiğiniz bir organizasyon.  
   ![Hesap seçin](/images/select-which-account-to-use.png)
6. Localizeflow’un erişmesini istediğiniz depoları seçin, ardından **Kaydet**'i tıklayın.  
   ![Depoyu seçin ve kaydedin](/images/select-repo-and-save.png)
7. Localizeflow ana sayfasına yönlendirileceksiniz.

> [!TIP]
> Daha sonra daha fazla depo eklemek için, üst menüde hesabınızı seçin ve **+ Daha fazla depo ekle** seçeneğini tıklayın.  
> ![Daha fazla depo ekle](/images/add-more-repo.png)

---

## Depolarınızı Localizeflow’a bağlayın

1. Localizeflow ana sayfasında **+ Depoları bağla** seçeneğini tıklayın.  
   ![Depoları bağla seçin](/images/select-connect-repos.png)

2. Bağlamak istediğiniz kurulu depolardan birini seçin ve **Kaydet**'i tıklayın.  
   ![Depoyu seçin](/images/select-repo.png)

3. Bağladığınız depolar artık hem Ana Sayfa hem de Depolar sayfasında görünecektir.  
   ![Bağlanan depolar](/images/repo-connected.png)

---

## Otomatik çeviriye başlayın

1. Az önce bağladığınız depoyu seçin.  
   ![Depoyu seçin](/images/select-repo-to-detail.png)

2. Depo detay sayfasında, sayfanın altındaki **Düzenle** seçeneğini tıklayın.  
   ![Düzenle seçin](/images/select-edit.png)

3. Çeviri ayarlarınızı yapılandırın — hedef dal (varsayılan: `main`), hedef diller ve kaynak dil (varsayılan: `en`). Ardından **Kaydet**'i seçin.  
   ![Çeviri ayarlarını yapılandırın](/images/configuration.png)

4. **Başlat & Otomatikleştir** seçeneğini tıklayın.  
   Localizeflow artık dokümantasyonunuzu otomatik olarak çevirecek ve kaynak değiştiğinde pull request açacaktır.  
   ![Başlat & Otomatikleştir](/images/select-automate.png)
