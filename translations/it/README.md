Localizeflow traduce automaticamente la tua documentazione e apre pull request ogni volta che il file sorgente viene modificato.  
Questa guida ti mostra come installare l’App GitHub e avviare la tua prima traduzione in meno di 2 minuti.

---

## Accedi e installa l’App GitHub

1. Visita **[localizeflow.com](https://localizeflow.com/)**.
2. Seleziona **Inizia con la prova gratuita**.  
   ![Seleziona Inizia con la prova gratuita](/images/select-start-with-free-trial.png)
3. Seleziona **Accedi con GitHub**.  
   ![Accedi con GitHub](/images/select-sign-in-with-github.png)
4. Accedi con il tuo account GitHub.  
   ![Login GitHub](/images/github-login.png)
5. Scegli l’account su cui vuoi installare l’App Localizeflow per GitHub — il tuo account personale o un’organizzazione che gestisci.  
   ![Seleziona account](/images/select-which-account-to-use.png)
6. Seleziona i repository a cui vuoi che Localizeflow abbia accesso, poi scegli **Salva**.  
   ![Seleziona repository e salva](/images/select-repo-and-save.png)
7. Verrai reindirizzato alla pagina principale di Localizeflow.

> [!TIP]
> Per aggiungere altri repository in seguito, seleziona il tuo account nell’intestazione e scegli **+ Aggiungi altri repository**.  
> ![Aggiungi altri repository](/images/add-more-repo.png)

---

## Collega i tuoi repository a Localizeflow

1. Nella pagina principale di Localizeflow, seleziona **+ Collega repository**.  
   ![Seleziona collega repository](/images/select-connect-repos.png)

2. Scegli uno dei repository installati che vuoi collegare e seleziona **Salva**.  
   ![Seleziona repository](/images/select-repo.png)

3. I repository collegati appariranno ora sia nella pagina Home che nella pagina Repository.  
   ![Repository collegati](/images/repo-connected.png)

---

## Avvia la traduzione automatica

1. Seleziona il repository che hai appena collegato.  
   ![Seleziona repository](/images/select-repo-to-detail.png)

2. Nella pagina dei dettagli del repository, seleziona **Modifica** in basso.  
   ![Seleziona modifica](/images/select-edit.png)

3. Configura le impostazioni di traduzione — branch di destinazione (predefinito: `main`), lingue di destinazione e lingua sorgente (predefinita: `en`). Seleziona **Salva**.  
   ![Configura impostazioni di traduzione](/images/configuration.png)

4. Seleziona **Avvia & Automatizza**.  
   Localizeflow tradurrà automaticamente la tua documentazione e aprirà pull request ogni volta che la sorgente cambia.  
   ![Avvia & Automatizza](/images/select-automate.png)
