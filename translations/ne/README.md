> **स्थानीय रूपमा क्लोन गर्न चाहनुहुन्छ?**

> यो रिपोजिटोरीमा ५०+ भाषा अनुवादहरू समावेश छन् जसले डाउनलोड आकार उल्लेखनीय रूपमा बढाउँछ। अनुवादहरू बिना क्लोन गर्न, sparse checkout प्रयोग गर्नुहोस्:  
> ```bash
> git clone --filter=blob:none --sparse https://github.com/localizeflow/localizeflow-docs.git
> cd localizeflow-docs
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
> यसले तपाईंलाई कोर्स पूरा गर्न आवश्यक सबै कुरा दिन्छ र छिटो डाउनलोड हुन्छ।

Localizeflow तपाईंको डकुमेन्टेशनलाई स्वचालित रूपमा अनुवाद गर्छ र स्रोत फाइल परिवर्तन हुँदा पर्सहरू खोल्छ।  
यो मार्गदर्शकले तपाईंलाई GitHub एप कसरी इन्स्टल गर्ने र २ मिनेट भित्र पहिलो अनुवाद कसरी चलाउने देखाउँछ।


> [!NOTE]
>
> Localizeflow हाल GitHub-आधारित डकुमेन्टेशन परियोजनाहरूलाई समर्थन गर्छ  
> (जस्तै: AI for Beginners र अधिकांश सामान्य खुल्ला स्रोत रिपोज)।  
> 
> Astro, Docusaurus, र Hugo जस्ता आधुनिक डकुमेन्टेशन फ्रेमवर्कको समर्थन  
> सक्रिय विकासमा छ।

## साइन इन गर्नुहोस् र GitHub एप इन्स्टल गर्नुहोस्

1. **[localizeflow.com](https://localizeflow.com/)** भ्रमण गर्नुहोस्।
2. **Start with free trial** छान्नुहोस्।  
   ![Start with free trial छान्नुहोस्](../../../../translated_images/ne/select-start-with-free-trial.6c2e287133ff9c8b.webp)
3. **Sign in with GitHub** छान्नुहोस्।  
   ![GitHub सँग साइन इन गर्नुहोस्](../../../../translated_images/ne/select-sign-in-with-github.f2850ffdd49cc894.webp)
4. आफ्नो GitHub खातामा साइन इन गर्नुहोस्।  
   ![GitHub लगइन](../../../../translated_images/ne/github-login.02bbaee0270b292e.webp)
5. जहाँ तपाईं Localizeflow GitHub एप इन्स्टल गर्न चाहनुहुन्छ त्यो खाता छान्नुहोस् — तपाईंको व्यक्तिगत खाता वा तपाईंले व्यवस्थापन गर्ने संस्था।  
   ![खाता छान्नुहोस्](../../../../translated_images/ne/select-which-account-to-use.7050f5ed0b773bb0.webp)
6. जस रिपोजिटोरीहरूमा Localizeflow पहुँच दिन चाहनुहुन्छ ती छान्नुहोस्, अनि **Save** छान्नुहोस्।  
   ![रिपो छान्नुहोस् र सेभ गर्नुहोस्](../../../../translated_images/ne/select-repo-and-save.5a95ae288aefec6e.webp)
7. तपाईंलाई Localizeflow गृहपृष्ठमा पुनःनेतृत्व गरिनेछ।

> [!TIP]
> पछि थप रिपोजिटोरीहरू थप्न, माथि आफ्नो खाता छान्नुहोस् र **+ Add more repositories** रोज्नुहोस्।  
> ![थप रिपोजिटोरीहरू थप्नुहोस्](../../../../translated_images/ne/add-more-repo.2ca5154473aaaafb.webp)

## आफ्नो रिपोजिटोरीहरूलाई Localizeflow सँग जोड्नुहोस्

1. Localizeflow गृहपृष्ठमा, **+ Connect repositories** छान्नुहोस्।  
   ![Connect repositories छान्नुहोस्](../../../../translated_images/ne/select-connect-repos.8ac6f96f77dcc62c.webp)

2. इन्स्टल भएको रिपोजिटोरीहरू मध्ये एउटा छान्नुहोस् र **Save** छान्नुहोस्।  
   ![रिपोजिटोरी छान्नुहोस्](../../../../translated_images/ne/select-repo.dce94db722b44cf3.webp)

3. तपाईंका जडान गरिएका रिपोजिटोरीहरू अब गृहपृष्ठ र रिपोजिटोरीहरू पृष्ठ दुवैमा देखिने छन्।  
   ![जडान गरिएका रिपोजिटोरीहरू](../../../../translated_images/ne/repo-connected.9e5c21ee789fdcaa.webp)

## स्वचालित अनुवाद सुरु गर्नुहोस्

1. हालै जडान गरेको रिपोजिटोरी छान्नुहोस्।  
   ![रिपोजिटोरी छान्नुहोस्](../../../../translated_images/ne/select-repo-to-detail.55e53233531f8518.webp)

2. रिपोजिटोरी विवरण पृष्ठमा तल **Edit** छान्नुहोस्।  
   ![Edit छान्नुहोस्](../../../../translated_images/ne/select-edit.225184c8c46d7001.webp)

3. अनुवाद सेटिङहरू कन्फिगर गर्नुहोस् — लक्ष्य शाखा (पूर्वनिर्धारित: `main`), लक्ष्य भाषा, र स्रोत भाषा (पूर्वनिर्धारित: `en`)। **Save** छान्नुहोस्।  
   ![अनुवाद सेटिङहरू](../../../../translated_images/ne/configuration.ab55d0f8bab5711b.webp)

4. **Start & Automate** छान्नुहोस्।  
   Localizeflow अब स्वचालित रूपमा तपाईंको डकुमेन्टेशन अनुवाद गर्नेछ र स्रोत परिवर्तन हरेक पटक पर्सहरू खोल्नेछ।  
   ![Start & Automate](../../../../translated_images/ne/select-automate.ff50c8d913b35a03.webp)
