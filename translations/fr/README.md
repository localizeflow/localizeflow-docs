Localizeflow traduit automatiquement votre documentation et ouvre des pull requests à chaque modification du fichier source.  
Ce guide vous montre comment installer l’application GitHub et lancer votre première traduction en moins de 2 minutes.

---

## Connectez-vous et installez l’application GitHub

1. Rendez-vous sur **[localizeflow.com](https://localizeflow.com/)**.
2. Sélectionnez **Commencer avec l’essai gratuit**.  
   ![Sélectionnez Commencer avec l’essai gratuit](/images/select-start-with-free-trial.png)
3. Choisissez **Se connecter avec GitHub**.  
   ![Se connecter avec GitHub](/images/select-sign-in-with-github.png)
4. Connectez-vous avec votre compte GitHub.  
   ![Connexion GitHub](/images/github-login.png)
5. Choisissez le compte sur lequel vous souhaitez installer l’application Localizeflow — votre compte personnel ou une organisation que vous gérez.  
   ![Sélectionnez le compte](/images/select-which-account-to-use.png)
6. Sélectionnez les dépôts auxquels vous voulez que Localizeflow accède, puis cliquez sur **Enregistrer**.  
   ![Sélectionnez le dépôt et enregistrez](/images/select-repo-and-save.png)
7. Vous serez redirigé vers la page d’accueil de Localizeflow.

> [!TIP]
> Pour ajouter d’autres dépôts plus tard, cliquez sur votre compte dans l’en-tête et choisissez **+ Ajouter d’autres dépôts**.  
> ![Ajouter d’autres dépôts](/images/add-more-repo.png)

---

## Connectez vos dépôts à Localizeflow

1. Sur la page d’accueil de Localizeflow, cliquez sur **+ Connecter des dépôts**.  
   ![Sélectionnez connecter des dépôts](/images/select-connect-repos.png)

2. Choisissez un des dépôts installés que vous souhaitez connecter, puis cliquez sur **Enregistrer**.  
   ![Sélectionnez le dépôt](/images/select-repo.png)

3. Vos dépôts connectés apparaîtront désormais à la fois sur la page d’accueil et sur la page des dépôts.  
   ![Dépôts connectés](/images/repo-connected.png)

---

## Lancez la traduction automatique

1. Sélectionnez le dépôt que vous venez de connecter.  
   ![Sélectionnez le dépôt](/images/select-repo-to-detail.png)

2. Sur la page de détails du dépôt, cliquez sur **Modifier** en bas.  
   ![Sélectionnez modifier](/images/select-edit.png)

3. Configurez vos paramètres de traduction — branche cible (par défaut : `main`), langues cibles, et langue source (par défaut : `en`). Cliquez sur **Enregistrer**.  
   ![Configurer les paramètres de traduction](/images/configuration.png)

4. Cliquez sur **Démarrer & Automatiser**.  
   Localizeflow traduira automatiquement votre documentation et ouvrira des pull requests à chaque modification de la source.  
   ![Démarrer & Automatiser](/images/select-automate.png)
