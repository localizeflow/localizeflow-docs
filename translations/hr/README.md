Localizeflow automatski prevodi vašu dokumentaciju i otvara pull requestove kad god se izvorna datoteka promijeni.  
Ovaj vodič pokazuje kako instalirati GitHub aplikaciju i pokrenuti prvi prijevod za manje od 2 minute.

---

## Prijavite se i instalirajte GitHub aplikaciju

1. Posjetite **[localizeflow.com](https://localizeflow.com/)**.
2. Odaberite **Start with free trial**.  
   ![Select Start with free trial](/images/select-start-with-free-trial.png)
3. Odaberite **Sign in with GitHub**.  
   ![Sign in with GitHub](/images/select-sign-in-with-github.png)
4. Prijavite se sa svojim GitHub računom.  
   ![GitHub login](/images/github-login.png)
5. Odaberite račun na koji želite instalirati Localizeflow GitHub aplikaciju — svoj osobni račun ili organizaciju kojom upravljate.  
   ![Select account](/images/select-which-account-to-use.png)
6. Odaberite repozitorije kojima želite da Localizeflow pristupi, zatim kliknite **Save**.  
   ![Select repo and save](/images/select-repo-and-save.png)
7. Bit ćete preusmjereni na početnu stranicu Localizeflow.

> [!TIP]
> Za dodavanje dodatnih repozitorija kasnije, odaberite svoj račun u zaglavlju i kliknite **+ Add more repositories**.  
> ![Add more repositories](/images/add-more-repo.png)

---

## Povežite svoje repozitorije s Localizeflow

1. Na početnoj stranici Localizeflow odaberite **+ Connect repositories**.  
   ![Select connect repositories](/images/select-connect-repos.png)

2. Odaberite jedan od instaliranih repozitorija koje želite povezati i kliknite **Save**.  
   ![Select repository](/images/select-repo.png)

3. Vaši povezani repozitoriji sada će se prikazivati i na početnoj stranici i na stranici Repositories.  
   ![Connected repositories](/images/repo-connected.png)

---

## Pokrenite automatski prijevod

1. Odaberite repozitorij koji ste upravo povezali.  
   ![Select repository](/images/select-repo-to-detail.png)

2. Na stranici s detaljima repozitorija kliknite **Edit** pri dnu.  
   ![Select edit](/images/select-edit.png)

3. Konfigurirajte postavke prijevoda — ciljnu granu (zadano: `main`), ciljne jezike i izvorni jezik (zadano: `en`). Kliknite **Save**.  
   ![Configure translation settings](/images/configuration.png)

4. Kliknite **Start & Automate**.  
   Localizeflow će sada automatski prevoditi vašu dokumentaciju i otvarati pull requestove kad god se izvor promijeni.  
   ![Start & Automate](/images/select-automate.png)
